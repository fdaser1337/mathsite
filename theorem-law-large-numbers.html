Закон больших чисел

Выборочное среднее сходится к математическому ожиданию при увеличении числа испытаний.

Формулировка

Пусть X₁, X₂, ..., Xₙ - независимые одинаково распределенные случайные величины с M[Xᵢ] = μ и D[Xᵢ] = σ² < ∞. Тогда выборочное среднее:

\[\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i\]

сходится к математическому ожиданию μ при n → ∞.

Слабый закон больших чисел

Для любого ε > 0:

\[\lim_{n \to \infty} P\left(|\bar{X}_n - \mu| \geq \varepsilon\right) = 0\]

Это сходимость по вероятности.

Усиленный закон больших чисел

С вероятностью 1:

\[P\left(\lim_{n \to \infty} \bar{X}_n = \mu\right) = 1\]

Это сходимость почти наверное (сходимость с вероятностью 1).

Интуитивное объяснение

При большом количестве наблюдений случайные колебания взаимно компенсируются, и среднее значение стабилизируется около истинного математического ожидания.

Аналогия: Если вы много раз подбрасываете монету, доля орлов будет стремиться к 0.5, даже если в коротких сериях могут быть значительные отклонения.

Доказательство слабого закона

Используем неравенство Чебышёва:

\[P\left(|\bar{X}_n - \mu| \geq \varepsilon\right) \leq \frac{D[\bar{X}_n]}{\varepsilon^2}\]

Так как M[\bar{X}_n] = μ и D[\bar{X}_n] = σ²/n:

\[P\left(|\bar{X}_n - \mu| \geq \varepsilon\right) \leq \frac{\sigma^2}{n\varepsilon^2} \to 0 \text{ при } n \to \infty\]

Примеры

Пример 1: Подбрасывание монеты

Xᵢ = 1, если выпал орёл, Xᵢ = 0, если выпала решка.

M[Xᵢ] = 0.5, D[Xᵢ] = 0.25.

Доля орлов в n подбрасываниях сходится к 0.5.

Пример 2: Измерение роста

При измерении роста n людей средний рост будет близок к истинному среднему росту населения.

Пример 3: Контроль качества

При проверке большой партии товаров доля брака сходится к истинной доле брака.

Условия применимости

1. Независимость

Случайные величины должны быть независимы.

2. Одинаковое распределение

Все величины должны иметь одинаковое распределение.

3. Конечная дисперсия

Математическое ожидание и дисперсия должны существовать и быть конечными.

Применения

1. Статистика

Обоснование выборочного метода, доверительные интервалы.

2. Экономика

Анализ рынков, страхование, управление рисками.

3. Физика

Статистическая механика, термодинамика.

4. Инженерия

Контроль качества, надёжность систем.
