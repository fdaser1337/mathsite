Теорема о сходимости опорных векторов

Сходимость метода опорных векторов и обобщающая способность SVM.

Основная постановка SVM

Задача классификации: даны обучающие данные {(x_i, y_i)}_{i=1}^n, где x_i ∈ ℝ^d, y_i ∈ {-1, +1}.

Линейно разделимый случай

Ищем разделяющую гиперплоскость w^T x + b = 0 с максимальным зазором:

\[\min_{w,b} \frac{1}{2}\|w\|^2\] \[\text{при условии: } y_i(w^T x_i + b) \geq 1, \quad i = 1, ..., n\]

Нелинейный случай

Используем ядро K(x_i, x_j) = φ(x_i)^T φ(x_j) для отображения в пространство признаков.

Двойственная задача

Формулировка

\[\max_{\alpha} \sum_{i=1}^{n} \alpha_i - \frac{1}{2}\sum_{i,j=1}^{n} \alpha_i \alpha_j y_i y_j K(x_i, x_j)\] \[\text{при условии: } 0 \leq \alpha_i \leq C, \quad \sum_{i=1}^{n} \alpha_i y_i = 0\]

где α_i - двойственные переменные, C - параметр регуляризации.

Решение

\[w = \sum_{i=1}^{n} \alpha_i y_i x_i\] \[b = y_k - w^T x_k \text{ для любого опорного вектора}\]

Теорема об обобщающей способности

Граница ошибки

Для SVM с ядром K:

\[R(h) \leq R_{emp}(h) + \sqrt{\frac{h(\log(2n/h) + 1) - \log(\eta/4)}{n}}\]

где R(h) - истинная ошибка, R_{emp}(h) - эмпирическая ошибка, h - VC-размерность.

VC-размерность SVM

Для SVM с ядром K:

\[h \leq \min\left(R^2\|w\|^2 + 1, n\right)\]

где R - радиус минимальной сферы, содержащей все данные.

Теорема о сходимости

Состоятельность

При n → ∞ и подходящем выборе параметра C_n:

\[R(\hat{f}_n) \to R(f^*)\]

где \hat{f}_n - SVM классификатор, f^* - Байесовский классификатор.

Условия сходимости

  • C_n → ∞, но C_n/n → 0
  • Ядро K универсально
  • Распределение данных имеет плотность

Опорные векторы

Определение

Точки x_i, для которых α_i > 0, называются опорными векторами.

Свойства

  • Опорные векторы лежат на границе зазора или нарушают его
  • Количество опорных векторов обычно меньше n
  • Решение зависит только от опорных векторов

Геометрическая интерпретация

Опорные векторы определяют разделяющую гиперплоскость.

Ядра SVM

Условия Мерсера

Функция K(x,y) является ядром, если она симметрична и положительно определена.

Популярные ядра

  • Линейное: K(x,y) = x^T y
  • Полиномиальное: K(x,y) = (x^T y + c)^d
  • RBF: K(x,y) = exp(-γ||x-y||²)
  • Сигмоидное: K(x,y) = tanh(ax^T y + c)

Алгоритмы обучения

SMO (Sequential Minimal Optimization)

Итеративная оптимизация двух переменных α_i, α_j за раз.

Градиентные методы

Прямая оптимизация исходной задачи с использованием градиентного спуска.

Координатный спуск

Покоординатная оптимизация двойственной задачи.

Многоклассовый SVM

Один-против-всех

Строим K бинарных классификаторов, каждый разделяет один класс от всех остальных.

Один-против-одного

Строим K(K-1)/2 бинарных классификаторов для каждой пары классов.

Прямой метод

Минимизация одной многоклассовой задачи оптимизации.

Регрессия SVM

ε-SVR

Вводим ε-нечувствительную функцию потерь:

\[L_\varepsilon(y, f(x)) = \max(0, |y - f(x)| - \varepsilon)\]

Двойственная задача

\[\max_{\alpha,\alpha^*} -\frac{1}{2}\sum_{i,j}(\alpha_i - \alpha_i^*)(\alpha_j - \alpha_j^*)K(x_i, x_j) - \varepsilon\sum_{i}(\alpha_i + \alpha_i^*) + \sum_{i}y_i(\alpha_i - \alpha_i^*)\]

Связь с другими методами

Логистическая регрессия

SVM максимизирует зазор, логистическая регрессия максимизирует правдоподобие.

Перцептрон

SVM - обобщение перцептрона с оптимальным зазором.

Байесовские классификаторы

SVM минимизирует структурный риск, Байесовские - апостериорный риск.

Практические аспекты

Выбор параметров

  • C - баланс между зазором и ошибками классификации
  • γ (для RBF) - ширина ядра
  • ε (для регрессии) - ширина трубки

Масштабирование признаков

Важно нормализовать или стандартизировать признаки перед обучением.

Вычислительная сложность

  • Обучение: O(n²) - O(n³) в зависимости от алгоритма
  • Классификация: O(количество опорных векторов)

Теоретические гарантии

Максимальный зазор

SVM находит гиперплоскость с максимальным зазором, что обеспечивает хорошую обобщающую способность.

Устойчивость к выбросам

Параметр C контролирует влияние выбросов на решение.

Выпуклая оптимизация

Задача SVM выпукла, что гарантирует нахождение глобального оптимума.

Применения

1. Биоинформатика

Классификация генов, анализ белков.

2. Распознавание образов

Распознавание лиц, текста, речи.

3. Финансы

Кредитный скоринг, предсказание временных рядов.

4. Медицина

Диагностика заболеваний, анализ медицинских изображений.

Историческая справка

Основан на теории статистического обучения Вапника-Червоненкиса.

Стал популярным благодаря хорошей производительности на многих задачах.
