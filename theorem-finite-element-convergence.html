Теорема о сходимости метода конечных элементов

Сходимость приближённых решений к точному решению дифференциальных уравнений.

Основная постановка

Рассмотрим эллиптическую краевую задачу:

\[-\nabla \cdot (A(x)\nabla u) + c(x)u = f(x) \text{ в } \Omega\] \[u = 0 \text{ на } \partial\Omega\]

где Ω ⊂ ℝⁿ - ограниченная область, A(x) - симметричная положительно определённая матрица.

Слабая формулировка: найти u ∈ H₀¹(Ω) такое, что:

\[a(u,v) = (f,v) \quad \forall v \in H_0^1(\Omega)\]

где a(u,v) = ∫Ω (A∇u·∇v + cuv) dx, (f,v) = ∫Ω fv dx.

Теорема Лакса-Мильграма

Формулировка

Пусть a(·,·) - билинейная форма на гильбертовом пространстве H, удовлетворяющая:

  1. Ограниченность: |a(u,v)| ≤ M‖u‖‖v‖
  2. Коэрцитивность: a(u,u) ≥ α‖u‖²

Тогда для любого линейного функционала F ∈ H* существует единственное решение u ∈ H уравнения a(u,v) = F(v) для всех v ∈ H.

Применение к МКЭ

Для нашей задачи a(u,v) ограничена и коэрцитивна, поэтому существует единственное решение.

Метод конечных элементов

Приближённая задача

Выбираем конечномерное подпространство Vₕ ⊂ H₀¹(Ω). Находим uₕ ∈ Vₕ такое, что:

\[a(u_h,v_h) = (f,v_h) \quad \forall v_h \in V_h\]

Примеры конечных элементов

  • Линейные элементы (P1): кусочно-линейные функции
  • Квадратичные элементы (P2): кусочно-квадратичные функции
  • Элементы Лагранжа: полиномы степени k

Теорема Сеза (Céa's Lemma)

Формулировка

Пусть u - точное решение, uₕ - приближённое решение МКЭ. Тогда:

\[\|u - u_h\|_H \leq \frac{M}{\alpha} \inf_{v_h \in V_h} \|u - v_h\|_H\]

где M - константа ограниченности, α - константа коэрцитивности.

Следствие

Ошибка МКЭ не более чем в M/α раз превосходит ошибку наилучшего приближения.

Аппроксимационные свойства

Теорема аппроксимации

Пусть u ∈ H^{k+1}(Ω), Vₕ состоит из полиномов степени k. Тогда:

\[\inf_{v_h \in V_h} \|u - v_h\|_{H^1} \leq Ch^k |u|_{H^{k+1}}\]

где h - максимальный размер элемента, C - константа, зависящая от формы элементов.

Скорость сходимости

Из теоремы Сеза и теоремы аппроксимации:

\[\|u - u_h\|_{H^1} \leq Ch^k |u|_{H^{k+1}}\]

Примеры сходимости

Пример 1: Уравнение Пуассона

-Δu = f в Ω, u = 0 на ∂Ω.

Для линейных элементов (k=1):

\[\|u - u_h\|_{H^1} \leq Ch \|u\|_{H^2}\]

Пример 2: Стационарное уравнение теплопроводности

-∇·(k∇u) = f, k > 0.

Для квадратичных элементов (k=2):

\[\|u - u_h\|_{H^1} \leq Ch^2 \|u\|_{H^3}\]

Сходимость в L²-норме

Теорема Обенжа-Ничче

Если a(·,·) симметрична, то:

\[\|u - u_h\|_{L^2} \leq Ch^{k+1} \|u\|_{H^{k+1}}\]

Сходимость в L² на порядок выше, чем в H¹.

Метод двойственности Обенжа

Позволяет получать оценки в L² для несимметричных задач.

Адаптивные методы

Идея адаптации

Измельчаем сетку там, где ошибка велика.

Апостериорные оценки ошибки

\[\|u - u_h\|_{H^1}^2 \leq C \sum_{K \in \mathcal{T}_h} h_K^2 \|R_K\|_{L^2(K)}^2 + \sum_{e \in \mathcal{E}_h} h_e \|J_e\|_{L^2(e)}^2\]

где R_K - остаток в элементе K, J_e - скачок производной на грани e.

Оптимальная сходимость

Адаптивные методы достигают оптимальной скорости сходимости.

Высокие порядки сходимости

p-метод МКЭ

Фиксированная сетка, увеличиваем степень полиномов p.

hp-метод МКЭ

Одновременно измельчаем сетку (h) и увеличиваем степень (p).

Экспоненциальная сходимость

Для гладких решений hp-метод даёт экспоненциальную сходимость.

Численные аспекты

Условия на сетку

  • Регулярность: соотношение размеров соседних элементов
  • Квази-однородность: ограничение на форму элементов

Вычислительная сложность

  • Сборка матрицы жёсткости: O(N)
  • Решение СЛАУ: O(N^1.5) для 2D, O(N²) для 3D
  • Многосеточные методы: O(N)

Применения

1. Инженерный анализ

Прочностные расчёты, анализ напряжений.

2. Гидродинамика

Уравнения Навье-Стокса, турбулентные течения.

3. Электромагнетизм

Уравнения Максвелла, расчёт полей.

4. Геофизика

Уравнения теплопроводности, фильтрации.
