Дифференциальные уравнения

Дифференциальные уравнения и методы их решения

Теория и практика решения дифференциальных уравнений: от простых ОДУ первого порядка до систем и уравнений в частных производных.

ОДУ первого порядка

Уравнения с разделяющимися переменными, однородные, линейные, в полных дифференциалах.

ОДУ высших порядков

Линейные уравнения с постоянными коэффициентами, метод вариации произвольных постоянных.

Системы ОДУ

Линейные системы, фазовые портреты, устойчивость, теория устойчивости Ляпунова.

Дифференциальные уравнения первого порядка

Уравнения с разделяющимися переменными

Общий вид:

\[M(x)dx + N(y)dy = 0\]

Решение:

\[\int M(x)dx + \int N(y)dy = C\]

Однородные уравнения

Уравнение вида $y' = f\left(\frac{y}{x}\right)$

Замена: $y = ux$, где $u = \frac{y}{x}$

\[\frac{dy}{dx} = u + x\frac{du}{dx}\]

Линейные уравнения первого порядка

Общий вид:

\[y' + P(x)y = Q(x)\]

Интегрирующий множитель:

\[\mu(x) = e^{\int P(x)dx}\]

Решение:

\[y = \frac{1}{\mu(x)}\left(\int \mu(x)Q(x)dx + C\right)\]

Уравнение Бернулли

Вид:

\[y' + P(x)y = Q(x)y^n\]

При $n \neq 0, 1$ заменой $z = y^{1-n}$ сводится к линейному уравнению.

Дифференциальные уравнения высших порядков

Линейные однородные уравнения с постоянными коэффициентами

Общий вид:

\[y^{(n)} + a_{n-1}y^{(n-1)} + \ldots + a_1y' + a_0y = 0\]

Характеристическое уравнение:

\[r^n + a_{n-1}r^{n-1} + \ldots + a_1r + a_0 = 0\]

Метод неопределённых коэффициентов

Для неоднородного уравнения $Ly = f(x)$

Если $f(x) = P_m(x)e^{\alpha x}\cos(\beta x) + Q_m(x)e^{\alpha x}\sin(\beta x)$

Тогда частное решение ищется в виде:

\[y_p = x^s \cdot R_m(x)e^{\alpha x}\cos(\beta x) + x^s \cdot S_m(x)e^{\alpha x}\sin(\beta x)\]

где $s$ - кратность корня $\alpha + i\beta$ характеристического уравнения.

Метод вариации произвольных постоянных

Для уравнения $Ly = f(x)$ с известным фундаментальным решением $y_1, y_2, \ldots, y_n$

Ищем решение в виде:

\[y = \sum_{i=1}^{n} C_i(x)y_i\]

С условиями:

\[\sum_{i=1}^{n} C_i'(x)y_i^{(k)} = 0, \quad k = 0, 1, \ldots, n-2\]

Системы дифференциальных уравнений

Линейные системы с постоянными коэффициентами

В матричной форме:

\[\frac{d\vec{x}}{dt} = A\vec{x}\]

Решение через экспоненту матрицы:

\[\vec{x}(t) = e^{At}\vec{x}_0\]

Фазовые портреты

Типы особых точек в зависимости от собственных значений:

  • Узел: действительные собственные значения одного знака
  • Седло: действительные собственные значения разных знаков
  • Фокус: комплексные сопряжённые с ненулевой действительной частью
  • Центр: чисто мнимые собственные значения

Теория устойчивости

Устойчивость по Ляпунову:

Решение $\vec{x}_0(t)$ устойчиво, если для любого $\varepsilon > 0$ существует $\delta > 0$ такое, что

\[\|\vec{x}(0) - \vec{x}_0(0)\| < \delta \Rightarrow \|\vec{x}(t) - \vec{x}_0(t)\| < \varepsilon\]

Асимптотическая устойчивость: устойчивость + сходимость при $t \to \infty$

Краевые задачи

Задача Штурма-Лиувилля

Общий вид:

\[\frac{d}{dx}\left[p(x)\frac{dy}{dx}\right] + [\lambda w(x) - q(x)]y = 0\]

с граничными условиями:

\[\alpha_1 y(a) + \alpha_2 y'(a) = 0\] \[\beta_1 y(b) + \beta_2 y'(b) = 0\]

Метод Фурье

Для уравнения теплопроводности:

\[\frac{\partial u}{\partial t} = a^2 \frac{\partial^2 u}{\partial x^2}\]

Решение в виде ряда Фурье:

\[u(x,t) = \sum_{n=1}^{\infty} C_n e^{-a^2\lambda_n t} X_n(x)\]
