Свойства функции

Каждая функции обладает областью определения, область значений, Единственность значения.

Область определения:

Множество х, то есть область допустимых значений выражения, которое записано в формуле.

Область значений:

Множество у, то есть это значения, которые может принимать функция.

Единственность значения:

Каждой совокупности аргументов соответствует ровно одно значение функции.

Виды функций

Постоянная функция

Задана формулой у = b, где b — некоторое число. Графиком постоянной функции является прямая, параллельная оси абсцисс и проходящая через точку (0; b) на оси ординат.

Прямая пропорциональность

Задана формулой у = kx, где коэффициент пропорциональности k ≠ 0. Графиком прямой пропорциональности является прямая, проходящая через начало координат.

Линейная функция

Задана формулой у = kx + b, где k и b — действительные числа. Графиком линейной функции является прямая.

Обратная пропорциональность

Задана формулой у = k/x, где k ≠ 0. K называется коэффициентом обратной пропорциональности. Графиком обратной пропорциональности является гипербола.

Степенная функция с натуральным показателем

Задана формулой у = хn, где n — натуральное число. Графики степенной функции с натуральным показателем зависят от n.

Степенная функция с целым отрицательным показателем

Представлена формулой у = х-n, где n — натуральное число. Данная функция определена при всех х ≠ 0.

Степенная функция с положительным дробным показателем

Представлена формулой у = хr, где r — положительная несократимая дробь. Данная функция также не является ни чётной, ни нечётной.

Логарифмическая функция

Функция, которая является обратной к операции возведения в степень.

Логарифм числа b по основанию a — это показатель степени с основанием a, равной b. Другими словами, логарифм — это степень, в которую нужно возвести a для получения b.

Некоторые свойства логарифмических функций:

Область определения — множество всех положительных чисел, то есть промежуток (0; + ∞).

Областью значения логарифмической функции является множество всех действительных чисел, то есть промежуток (– ∞; + ∞)

Логарифмическая функция является строго монотонной: при основании а > 1 она возрастает, а при основании 0

Тригонометрические функции

Функция синуса

Элементарные функции, которые исторически возникли при рассмотрении прямоугольных треугольников и выражали зависимости длин сторон этих треугольников от острых углов при гипотенузе

Периодичность: функция периодическая с минимальным периодом 2π. Это означает, что значение функции начинает повторяться через интервал в 2π

Нечётность: sin(-x) = -sin(x). График функции симметричен относительно начала координат.

Область значений: [-1; 1]. Зависимая переменная принимает все значения от -1 до 1.

Область определения: все числа. Функция определена на множестве действительных чисел, то есть аргумент x может быть любым действительным числом.

Непрерывность: функция непрерывна на всей области определения.

Ограниченность: график функции лежит в полосе, ограниченной прямыми y = 1 и y = -1.

Функция косинуса

Элементарная тригонометрическая функция, которая выражает зависимость угла при вершине треугольника от отношения прилежащей его стороны к гипотенузе.

Область определения — множество всех действительных чисел.

Область значений — отрезок от −1 до 1.

Функция чётная, то есть cos(−x) = cos(x).

Функция периодическая, её период равен 2π.

Максимальные значения функции достигаются в точках x = 2πk, где k — любое целое число.

Минимальные значения функции достигаются в точках x = π/2 + 2πk.

Функция возрастает на отрезках -π + 2πk ≤ x ≤ 2πk, убывает — на отрезках 2πk ≤ x ≤ π + 2πk.

Функция непрерывна, то есть не имеет разрывов.

Функция тангенса

Одна из тригонометрических функций в математике. Тангенсом угла называют число, которое равно отношению синуса угла к косинусу данного угла.

Область определения: все действительные числа, кроме точек, в которых косинус принимает нулевые значения.

Область значений: все числа.

Нечётность: функция тангенса нечётная, её график симметричен относительно начала координат.

Периодичность: функция тангенса периодическая, минимальный период равен π.

Возрастание: функция тангенса возрастает на всей области определения.

Отсутствие ограничений: функция тангенса не ограничена ни снизу, ни сверху, не имеет ни наибольшего, ни наименьшего значений.

Точка разрыва второго рода: функция тангенса имеет точку разрыва второго рода при определённых значениях аргумента.

Функция котангенса

Тригонометрическая функция острого угла в прямоугольном треугольнике, равная отношению прилежащего катета к противолежащему катету.

Область определения: вся числовая ось, кроме точек x = πn, где n — любое целое число.

Область значений: вся числовая ось.

Функция нечётная: ctg(−x) = −ctg x.

Функция периодическая: наименьший положительный период равен π.

Функция убывает: на каждом из промежутков вида (πn; π+πn), n — любое целое число.

Функция непрерывная: непрерывна на каждом из интервалов своей области определения.

Функция не ограничена ни снизу, ни сверху.

Монотонность функции

Монотонность функции в математике — это характеристика графика функции, описывающая его промежутки возрастания и убывания.

Функция возрастает, если она «поднимается» слева направо. Функция убывает, если она «падает» слева направо.

Если функция на всём промежутке только возрастает или только убывает, то говорят, что она монотонно возрастает или монотонно убывает.

Если функция на различных промежутках то убывает, то возрастает, то такую функцию называют немонотонной и описывают её возрастание и убывание на конкретных промежутках.

Производная

Производная функции — это предел отношения приращения функции к приращению её аргумента, при условии, что приращение аргумента стремится к нулю.

Простыми словами, производная функции описывает, как и с какой скоростью эта функция меняется в данной конкретной точке.

Физический смысл производной

Физический смысл производной — её интерпретация в контексте реальных процессов и явлений. Производная позволяет точно описать, как быстро изменяется одна физическая величина относительно другой во времени или в зависимости от иных параметров.

Геометрический смысл производной

Геометрический смысл производной заключается в том, что производная от функции в точке равна тангенсу угла между осью OX и касательной к графику функции в данной точке.

Экстремумы

Нахождение экстремумов: С помощью производной можно находить максимумы и минимумы функции. Это важно в оптимизации.
