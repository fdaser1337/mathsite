# Документация проекта MathLab

## Общее описание

MathLab — это образовательный веб-сайт, представляющий собой структурированный справочник по высшей математике для студентов университетов. Проект разработан как бесплатная цифровая платформа для систематизации математических знаний с удобной навигацией по теории и практике.

**Тип проекта**: Статический образовательный сайт  
**Лицензия**: GNU GPL v3  
**Стек технологий**: HTML, CSS, JavaScript (vanilla)  
**Целевая аудитория**: Студенты математических дисциплин (1-4 курс)

## Структура проекта


## Архитектура и технологии

- **HTML5**: Семантическая разметка, доступность

## Технические особенности

### CSS-архитектура
- **CSS Custom Properties**: Централизованное управление цветами и стилями
- **Модульность**: Разделение стилей на логические блоки
- **Адаптивность**: Гибкие сетки и медиа-запросы
- **Производительность**: Минимизация переопределений стилей

### JavaScript-функционал
- **Поиск**: Дебажированный поиск по индексу
- **Темизация**: Сохранение предпочтений в localStorage
- **Фильтрация**: Динамическое скрытие/показ карточек
- **Навигация**: Плавные переходы и hash-based routing

### SEO и доступность
- **Семантическая разметка**: Правильная структура HTML
- **Meta-теги**: Оптимизация для поисковых систем
- **ARIA-атрибуты**: Поддержка скринридеров
- **Keyboard navigation**: Навигация с клавиатуры

## Разработка и внесение изменений

### Стандарты контента
1. **Академичность**: Строгие формулировки и доказательства
2. **Единый формат**: Стандартизированная структура страниц
3. **Математическая точность**: Корректное использование формул
4. **Практическая направленность**: Примеры и применения

### Как добавить новую теорему
1. Создать HTML-файл с описанием теоремы
2. Добавить запись в `THEOREM_CATALOG` в `js/main.js`
3. Указать курс и хештеги для категоризации
4. Соблюдать единый стиль оформления

### Как улучшить существующий материал
1. Добавить качественные визуализации
2. Включить ссылки на проверенные лекции
3. Расширить раздел с примерами
4. Улучшить математическое оформление
