Численные методы и вычислительная математика

Практические методы решения математических задач: от решения уравнений до численного интегрирования и дифференцирования.

Метод Ньютона, метод бисекции, метод секущих, итерационные методы.

Полином Лагранжа, сплайны, метод наименьших квадратов, аппроксимация.

Формулы Ньютона-Котеса, метод Гаусса, адаптивные методы.

Численное решение уравнений

Метод бисекции (деления отрезка пополам)

Для решения $f(x) = 0$ на отрезке $[a, b]$ где $f(a) \cdot f(b) < 0$

\[x_{n+1} = \frac{a_n + b_n}{2}\]

Выбор нового отрезка:

\[f(a_n) \cdot f(x_{n+1}) < 0 \Rightarrow [a_{n+1}, b_{n+1}] = [a_n, x_{n+1}]\] \[f(a_n) \cdot f(x_{n+1}) > 0 \Rightarrow [a_{n+1}, b_{n+1}] = [x_{n+1}, b_n]\]

Скорость сходимости: линейная, погрешность уменьшается вдвое на каждой итерации.

Метод Ньютона (метод касательных)

Итерационная формула:

\[x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\]

Геометрическая интерпретация: пересечение касательной в точке $x_n$ с осью $Ox$

Скорость сходимости: квадратичная при достаточно хорошем начальном приближении.

Условие сходимости: $|f(x) \cdot f''(x)| < |f'(x)|^2$ на отрезке.

Метод секущих

Модификация метода Ньютона без вычисления производной:

\[x_{n+1} = x_n - f(x_n) \cdot \frac{x_n - x_{n-1}}{f(x_n) - f(x_{n-1})}\]

Преимущество: не требует вычисления производной.

Недостаток: требует двух начальных приближений.

Метод простой итерации

Преобразование уравнения $f(x) = 0$ к виду $x = \varphi(x)$

\[x_{n+1} = \varphi(x_n)\]

Достаточное условие сходимости: $|\varphi'(x)| \leq q < 1$ на отрезке.

Скорость сходимости: линейная с коэффициентом $q$.

Интерполяция и аппроксимация

Интерполяционный полином Лагранжа

Для узлов $(x_0, y_0), (x_1, y_1), \ldots, (x_n, y_n)$

\[L_n(x) = \sum_{i=0}^{n} y_i \cdot l_i(x)\]

где базисные полиномы:

\[l_i(x) = \prod_{\substack{j=0 \\ j \neq i}}^{n} \frac{x - x_j}{x_i - x_j}\]

Свойство: $L_n(x_i) = y_i$ для всех $i = 0, 1, \ldots, n$

Интерполяционный полином Ньютона

Разделённые разности:

\[f[x_i, x_{i+1}] = \frac{f(x_{i+1}) - f(x_i)}{x_{i+1} - x_i}\]
\[f[x_i, x_{i+1}, x_{i+2}] = \frac{f[x_{i+1}, x_{i+2}] - f[x_i, x_{i+1}]}{x_{i+2} - x_i}\]

Интерполяционная формула:

\[P_n(x) = f(x_0) + f[x_0, x_1](x - x_0) + f[x_0, x_1, x_2](x - x_0)(x - x_1) + \ldots\]

Кубические сплайны

Кусочно-полиномиальная интерпретация степени 3

На каждом отрезке $[x_i, x_{i+1}]$:

\[S_i(x) = a_i + b_i(x - x_i) + c_i(x - x_i)^2 + d_i(x - x_i)^3\]

Условия:

  • $S_i(x_i) = y_i$, $S_i(x_{i+1}) = y_{i+1}$ (интерполяция)
  • $S'_i(x_{i+1}) = S'_{i+1}(x_{i+1})$ (непрерывность первой производной)
  • $S''_i(x_{i+1}) = S''_{i+1}(x_{i+1})$ (непрерывность второй производной)

Метод наименьших квадратов

Аппроксимация данных $(x_i, y_i)$ функцией $f(x, \vec{a})$

\[\Phi(\vec{a}) = \sum_{i=1}^{n} [y_i - f(x_i, \vec{a})]^2 \to \min\]

Для линейной аппроксимации $y = ax + b$:

\[a = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{n\sum x_i^2 - (\sum x_i)^2}\] \[b = \frac{\sum y_i - a\sum x_i}{n}\]

Численное интегрирование

Формула трапеций

\[\int_a^b f(x)dx \approx \frac{h}{2} \left[ f(a) + 2\sum_{i=1}^{n-1} f(x_i) + f(b) \right]\]

где $h = \frac{b-a}{n}$, $x_i = a + ih$

Погрешность: $O(h^2)$

Формула Симпсона

\[\int_a^b f(x)dx \approx \frac{h}{3} \left[ f(x_0) + 4\sum_{i=1,3,5,\ldots}^{n-1} f(x_i) + 2\sum_{i=2,4,6,\ldots}^{n-2} f(x_i) + f(x_n) \right]\]

где $n$ - чётное, $h = \frac{b-a}{n}$

Погрешность: $O(h^4)$

Квадратурные формулы Гаусса

\[\int_{-1}^{1} f(x)dx \approx \sum_{i=1}^{n} A_i f(x_i)\]

где $x_i$ - корни полинома Лежандра $P_n(x)$

Веса:

\[A_i = \frac{2}{(1-x_i^2)[P'_n(x_i)]^2}\]

Точность: точна для полиномов степени до $2n-1$

Адаптивные методы

Принцип Ричардсона экстраполяции:

\[I \approx I_h + \frac{I_h - I_{2h}}{2^p - 1}\]

где $p$ - порядок точности метода

Автоматический выбор шага интегрирования для достижения заданной точности.

Численное решение систем линейных уравнений

Метод Гаусса

Прямой ход: приведение к треугольному виду

Обратный ход: нахождение решений

Сложность: $O(n^3)$ операций

Метод Якоби

Для системы $Ax = b$, итерационная формула:

\[x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j \neq i} a_{ij} x_j^{(k)} \right)\]

Условие сходимости: диагональное преобладание матрицы $A$

Метод Гаусса-Зейделя

\[x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j < i} a_{ij} x_j^{(k+1)} - \sum_{j > i} a_{ij} x_j^{(k)} \right)\]

Преимущество: обычно сходится быстрее метода Якоби
