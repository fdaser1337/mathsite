Теорема Грина

Связывает криволинейный интеграл по замкнутому контуру с двойным интегралом по области.

Формулировка

Пусть D - ограниченная область с кусочно-гладкой границей ∂D, ориентированной против часовой стрелки. Если функции P(x,y) и Q(x,y) имеют непрерывные частные производные в D, то:

\[\oint_{\partial D} P \, dx + Q \, dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dx \, dy\]

Геометрическая интерпретация

Теорема Грина - это двумерный аналог фундаментальной теоремы анализа. Она связывает интеграл по границе с интегралом по площади.

Левая часть - «циркуляция» поля вдоль границы, правая часть - «вихрь» поля внутри области.

Форма площади

Из теоремы Грина можно получить формулы для вычисления площади области:

\[\text{Площадь}(D) = \oint_{\partial D} x \, dy = -\oint_{\partial D} y \, dx = \frac{1}{2}\oint_{\partial D} x \, dy - y \, dx\]

Применения

1. Вычисление площадей

Площадь сложных фигур через контурный интеграл.

2. Электродинамика

Связь между электрическим полем и магнитным полем (уравнения Максвелла).

3. Гидродинамика

Расчёт циркуляции и вихрей в потоках жидкости.

4. Компьютерная графика

Расчёт освещения и теней в 2D-графике.

Пример вычисления

Вычислить \[\oint_C x^2 y \, dx + xy^2 \, dy\] где C - граница квадрата [0,1] × [0,1].

По теореме Грина:

\[\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = \frac{\partial}{\partial x}(xy^2) - \frac{\partial}{\partial y}(x^2 y) = y^2 - x^2\]
\[\iint_{[0,1]^2} (y^2 - x^2) \, dx \, dy = \int_0^1 \int_0^1 (y^2 - x^2) \, dx \, dy = \int_0^1 \left[y^2x - \frac{x^3}{3}\right]_0^1 dy = \int_0^1 \left(y^2 - \frac{1}{3}\right) dy = \left[\frac{y^3}{3} - \frac{y}{3}\right]_0^1 = 0\]

Обобщения

Формула Грина-Римана

Для комплексной функции f(z) = u(x,y) + iv(x,y):

\[\oint_C f(z) \, dz = 2i \iint_D \frac{\partial f}{\partial \bar{z}} \, dx \, dy\]
