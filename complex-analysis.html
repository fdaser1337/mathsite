Комплексный анализ и функции комплексного переменного

Теория функций комплексного переменного: от основ комплексных чисел до конформных отображений и теории вычетов.

Комплексные числа и их свойства

Алгебраическая форма

Комплексное число в алгебраической форме:

\[z = x + iy\]

где $x = \text{Re}(z)$ - действительная часть, $y = \text{Im}(z)$ - мнимая часть.

Операции:

\[z_1 + z_2 = (x_1 + x_2) + i(y_1 + y_2)\] \[z_1 \cdot z_2 = (x_1x_2 - y_1y_2) + i(x_1y_2 + x_2y_1)\]

Тригонометрическая форма

\[z = r(\cos\varphi + i\sin\varphi) = re^{i\varphi}\]

где $r = |z| = \sqrt{x^2 + y^2}$ - модуль, $\varphi = \arg(z) = \arctan\frac{y}{x}$ - аргумент.

Формула Муавра:

\[z^n = r^n(\cos n\varphi + i\sin n\varphi)\]

Корни из комплексных чисел

\[\sqrt[n]{z} = \sqrt[n]{r}\left[\cos\left(\frac{\varphi + 2k\pi}{n}\right) + i\sin\left(\frac{\varphi + 2k\pi}{n}\right)\right]\]

где $k = 0, 1, 2, \ldots, n-1$

Аналитические функции

Условия Коши-Римана

Для функции $f(z) = u(x,y) + iv(x,y)$ условия Коши-Римана:

\[\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}\]

В полярных координатах:

\[\frac{\partial u}{\partial r} = \frac{1}{r}\frac{\partial v}{\partial\varphi}, \quad \frac{\partial v}{\partial r} = -\frac{1}{r}\frac{\partial u}{\partial\varphi}\]

Голоморфные функции

Функция $f(z)$ называется голоморфной в области $D$, если она дифференцируема в каждой точке этой области.

Производная функции комплексного переменного:

\[f'(z) = \lim_{\Delta z \to 0} \frac{f(z + \Delta z) - f(z)}{\Delta z}\]

Степенные ряды

Ряд Тейлора для аналитической функции:

\[f(z) = \sum_{n=0}^{\infty} \frac{f^{(n)}(z_0)}{n!}(z - z_0)^n\]

Радиус сходимости:

\[R = \frac{1}{\limsup_{n \to \infty} \sqrt[n]{|a_n|}}\]

Ряд Лорана

\[f(z) = \sum_{n=-\infty}^{\infty} a_n(z - z_0)^n = \sum_{n=0}^{\infty} a_n(z - z_0)^n + \sum_{n=1}^{\infty} a_{-n}(z - z_0)^{-n}\]

где первая сумма - правильная часть, вторая - главная часть.

Интегрирование функций комплексного переменного

Интеграл Коши

\[\int_\gamma f(z)dz = \int_a^b f(\gamma(t))\gamma'(t)dt\]

где $\gamma: [a,b] \to \mathbb{C}$ - параметризация контура.

Интегральная теорема Коши

Если $f(z)$ аналитична в односвязной области $D$ и $\gamma \subset D$ - замкнутый контур, то:

\[\oint_\gamma f(z)dz = 0\]

Интегральная формула Коши

\[f(z_0) = \frac{1}{2\pi i}\oint_\gamma \frac{f(z)}{z - z_0}dz\]

Производные:

\[f^{(n)}(z_0) = \frac{n!}{2\pi i}\oint_\gamma \frac{f(z)}{(z - z_0)^{n+1}}dz\]

Вычеты

Вычет функции $f(z)$ в точке $z_0$:

\[\text{Res}(f, z_0) = \frac{1}{2\pi i}\oint_{|z-z_0|=\rho} f(z)dz\]

Для простого полюса:

\[\text{Res}(f, z_0) = \lim_{z \to z_0} (z - z_0)f(z)\]

Основная теорема о вычетах:

\[\oint_\gamma f(z)dz = 2\pi i \sum_{k} \text{Res}(f, z_k)\]

Конформные отображения

Определение конформного отображения

Отображение $w = f(z)$ называется конформным в точке $z_0$, если:

  • $f$ голоморфна в окрестности $z_0$
  • $f'(z_0) \neq 0$

Свойства: сохранение углов и локальное подобие.

Основные конформные отображения

Линейная функция: $w = az + b$

Обратная функция: $w = \frac{1}{z}$

Дробно-линейная функция: $w = \frac{az + b}{cz + d}$, $ad - bc \neq 0$

Степенная функция: $w = z^n$

Экспонента: $w = e^z$

Логарифм: $w = \ln z$

Принцип симметрии

Если $f(z)$ аналитична в области, симметричной относительно действительной оси, и принимает действительные значения на этой оси, то:

\[f(\bar{z}) = \overline{f(z)}\]

Специальные функции

Экспонента и логарифм

\[e^z = e^x(\cos y + i\sin y)\] \[\ln z = \ln|z| + i\arg(z)\]

Тригонометрические функции

\[\sin z = \frac{e^{iz} - e^{-iz}}{2i}\] \[\cos z = \frac{e^{iz} + e^{-iz}}{2}\]

Гиперболические функции

\[\sinh z = \frac{e^z - e^{-z}}{2}\] \[\cosh z = \frac{e^z + e^{-z}}{2}\]
