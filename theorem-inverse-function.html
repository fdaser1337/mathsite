Теорема об обратной функции

Условия существования обратной функции и её дифференцируемость.

Формулировка

Пусть функция f непрерывно дифференцируема на интервале (a,b) и f'(x) ≠ 0 для всех x ∈ (a,b). Тогда:

  1. f строго монотонна на (a,b)
  2. Существует обратная функция f⁻¹: f((a,b)) → (a,b)
  3. f⁻¹ непрерывно дифференцируема и:
\[(f^{-1})'(y) = \frac{1}{f'(x)} \quad \text{где} \quad y = f(x)\]

Геометрическая интерпретация

График обратной функции симметричен графику исходной функции относительно прямой y = x.

Условие f'(x) ≠ 0 означает, что касательная к графику никогда не горизонтальна. Это гарантирует, что функция локально обратима.

Аналогия: Если вы идёте по дороге без горизонтальных участков, вы всегда можете вернуться назад по тому же пути.

Многомерный случай

Для функции F: ℝⁿ → ℝⁿ с непрерывными частными производными:

Если матрица Якоби J_F(x) обратима в точке a (det J_F(a) ≠ 0), то:

  1. Существует окрестность U точки a, где F взаимно однозначна
  2. Обратная функция F⁻¹ дифференцируема и:
\[J_{F^{-1}}(F(a)) = J_F(a)^{-1}\]

Примеры

Пример 1: Экспонента

f(x) = eˣ, f'(x) = eˣ > 0 для всех x.

Теорема применима: существует обратная функция ln(y).

\[(\ln)'(y) = \frac{1}{f'(x)} = \frac{1}{e^x} = \frac{1}{y}\]

Пример 2: Кубическая функция

f(x) = x³, f'(x) = 3x².

В точке x = 0: f'(0) = 0, теорема неприменима.

На интервалах (-∞,0) и (0,∞): f'(x) > 0, теорема применима.

Пример 3: Тригонометрические функции

f(x) = sin(x) на [-π/2, π/2], f'(x) = cos(x) > 0.

Теорема применима: существует arcsin(y).

\[(\arcsin)'(y) = \frac{1}{\cos(x)} = \frac{1}{\sqrt{1-y^2}}\]

Важность условий

1. Непрерывная дифференцируемость

Функция должна быть гладкой без разрывов производной.

2. Ненулевая производная

f'(x) ≠ 0 гарантирует локальную обратимость.

3. Связность области

Интервал должен быть связным множеством.

Следствия

1. Монотонность

Если f' > 0, то f возрастает; если f' < 0, то f убывает.

2. Локальная обратимость

В окрестности точки с f'(a) ≠ 0 функция всегда обратима.

3. Дифференцирование обратных функций

Производная обратной функции равна обратной величине производной.

Применения

1. Решение уравнений

Метод Ньютона, численное решение нелинейных уравнений.

2. Теория оптимизации

Анализ условий оптимальности, метод Лагранжа.

3. Экономика

Функции спроса и предложения, эластичность.

4. Физика

Обратные преобразования, изменение переменных.
