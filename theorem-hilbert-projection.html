Теорема о проективности гильбертова пространства

Для любого замкнутого подпространства гильбертова пространства существует ортогональная проекция.

Формулировка

Пусть H - гильбертово пространство, M ⊂ H - замкнутое подпространство. Тогда:

  1. Для любого x ∈ H существует единственный элемент y₀ ∈ M такой, что ||x - y₀|| = dist(x,M)
  2. Элемент y₀ является ортогональной проекцией x на M
  3. Разложение x = y₀ + z, где z ∈ M⊥ (ортогональное дополнение)

Оператор P: H → M, P(x) = y₀ называется оператором ортогональной проекции на M.

Основные определения

Гильбертово пространство

Полное пространство со скалярным произведением ⟨·,·⟩.

Ортогональное дополнение

M⊥ = {z ∈ H : ⟨z,y⟩ = 0 для всех y ∈ M}.

Ортогональная проекция

Отображение P: H → M такое, что x - P(x) ⊥ M.

Доказательство существования

Шаг 1: Минимизирующая последовательность

Пусть d = dist(x,M) = inf{||x-y|| : y ∈ M}. Выберем последовательность {yₙ} ⊂ M такую, что ||x-yₙ|| → d.

Шаг 2: Сходимость последовательности

Используем параллелограммное тождество:

\[\|y_n - y_m\|^2 = 2\|x-y_n\|^2 + 2\|x-y_m\|^2 - 4\|x - \frac{y_n+y_m}{2}\|^2\]

Поскольку (yₙ+yₘ)/2 ∈ M, то ||x - (yₙ+yₘ)/2|| ≥ d. Отсюда:

\[\|y_n - y_m\|^2 \leq 2\|x-y_n\|^2 + 2\|x-y_m\|^2 - 4d^2 \to 0\]

Следовательно, {yₙ} - последовательность Коши в M, поэтому yₙ → y₀ ∈ M.

Шаг 3: Ортогональность

Для любого y ∈ M и λ ∈ ℝ:

\[\|x - (y_0 + \lambda y)\|^2 = \|x - y_0\|^2 - 2\lambda \text{Re}\langle x - y_0, y\rangle + \lambda^2\|y\|^2\]

Минимум достигается при λ = 0, поэтому ⟨x-y₀, y⟩ = 0 для всех y ∈ M.

Свойства оператора проекции

1. Линейность

P - линейный оператор: P(αx + βy) = αP(x) + βP(y).

2. Идемпотентность

P² = P (проекция проекции есть сама проекция).

3. Самосопряжённость

⟨Px, y⟩ = ⟨x, Py⟩ для всех x, y ∈ H.

4. Норма

||P|| = 1 (если M ≠ {0}).

Примеры

Пример 1: Прямая в ℝ²

H = ℝ², M = {(x,0) : x ∈ ℝ}. Проекция: P(x,y) = (x,0).

Пример 2: Подпространство в ℝ³

H = ℝ³, M = {(x,y,0) : x,y ∈ ℝ}. Проекция: P(x,y,z) = (x,y,0).

Пример 3: Пространство L²[0,1]

H = L²[0,1], M = {f ∈ L²[0,1] : f(t) = 0 для t ∈ [0,1/2]}.

Проекция: (Pf)(t) = f(t) для t ∈ (1/2,1], (Pf)(t) = 0 для t ∈ [0,1/2].

Пример 4: Пространство ℓ²

H = ℓ², M = {x = (x₁,x₂,0,0,…) : x₁,x₂ ∈ ℝ}.

Проекция: P(x₁,x₂,x₃,…) = (x₁,x₂,0,0,…).

Теорема о разложении

Как следствие, получаем ортогональное разложение:

\[H = M \oplus M^\perp\]

То есть каждый элемент x ∈ H единственным образом представляется в виде x = y + z, где y ∈ M, z ∈ M⊥.

Применения

1. Теория приближений

Наилучшее приближение элементами подпространства.

2. Квантовая механика

Измерение квантовых состояний, редукция волновой функции.

3. Обработка сигналов

Фильтрация, сжатие данных, анализ Фурье.

4. Машинное обучение

Метод наименьших квадратов, регрессия.

Связь с другими теоремами

Теорема Рисса об ортогональном дополнении

(M⊥)⊥ = M для замкнутого подпространства M.

Теорема Хана-Банаха

Продолжение линейных функционалов с сохранением нормы.

Теорема о базисе

Существование ортонормированного базиса в сепарабельном гильбертовом пространстве.

Обобщения

Банаховы пространства

В банаховых пространствах проекция существует не всегда.

Неортогональные проекции

Проекции на подпространства без условия ортогональности.

Метрические проекции

В метрических пространствах ближайшая точка может быть неединственной.
