Теорема о сходимости метода Монте-Карло

Статистическая сходимость Монте-Карло оценок к математическим ожиданиям.

Формулировка

Пусть X₁, X₂, ..., Xₙ - независимые одинаково распределённые случайные величины с математическим ожиданием μ и дисперсией σ² < ∞. Тогда оценка Монте-Карло:

\[\hat{\mu}_n = \frac{1}{n}\sum_{i=1}^{n} X_i\]

сходится к μ при n → ∞ в следующих смыслах:

  1. Почти наверное: P(lim_{n→∞} \hat{μ}_n = μ) = 1 (усиленный закон больших чисел)
  2. По вероятности: lim_{n→∞} P(|\hat{μ}_n - μ| > ε) = 0 для любого ε > 0
  3. В среднеквадратичном: E[(\hat{μ}_n - μ)²] = σ²/n → 0

Скорость сходимости

Центральная предельная теорема

По ЦПТ:

\[\sqrt{n}(\hat{\mu}_n - \mu) \xrightarrow{d} \mathcal{N}(0, \sigma^2)\]

Это означает, что скорость сходимости порядка O(1/√n).

Доверительные интервалы

Приближённый 95% доверительный интервал:

\[\hat{\mu}_n \pm 1.96\frac{s}{\sqrt{n}}\]

где s² - выборочная дисперсия.

Сравнение с детерминированными методами

  • Монте-Карло: O(1/√n) сходимость
  • Детерминированные методы: O(1/n) или лучше
  • Преимущество Монте-Карло: независимость от размерности

Доказательство сходимости

Сходимость в среднеквадратичном

\[E[(\hat{\mu}_n - \mu)^2] = \text{Var}(\hat{\mu}_n) = \frac{\sigma^2}{n}\]

Сходимость по вероятности

По неравенству Чебышёва:

\[P(|\hat{\mu}_n - \mu| > \varepsilon) \leq \frac{\text{Var}(\hat{\mu}_n)}{\varepsilon^2} = \frac{\sigma^2}{n\varepsilon^2} \to 0\]

Почти наверное сходимость

Следует из усиленного закона больших чисел Колмогорова.

Применения для интегрирования

Вычисление определённых интегралов

Для I = ∫_a^b f(x)dx:

\[I = (b-a)E[f(U)]\]

где U ~ Uniform(a,b). Оценка:

\[\hat{I}_n = \frac{b-a}{n}\sum_{i=1}^{n} f(U_i)\]

Многомерные интегралы

Для I = ∫_D f(x)dx в d-мерной области:

\[\hat{I}_n = \frac{\text{Vol}(D)}{n}\sum_{i=1}^{n} f(X_i)\]

где X_i ~ Uniform(D). Скорость сходимости не зависит от d!

Примеры

Пример 1: Вычисление π

Используем метод бросания иглы:

\[P(\text{точка в круге}) = \frac{\pi}{4}\]

Оценка: \hat{π}_n = 4·(количество точек в круге)/n.

Пример 2: Интеграл от sin(x)

I = ∫_0^π sin(x)dx = 2.

Монте-Карло оценка: \hat{I}_n = π·(1/n)∑sin(U_i), где U_i ~ Uniform(0,π).

Пример 3: Многомерный интеграл

I = ∫_[0,1]^d exp(-∑x_i²)dx.

Монте-Карло работает для любого d, детерминированные методы страдают от проклятия размерности.

Улучшение сходимости

Метод выборки по значимости

\[I = \int f(x)dx = \int \frac{f(x)}{g(x)}g(x)dx\]

Выбираем g(x) ≈ |f(x)| для уменьшения дисперсии.

Метод антитезических переменных

Используем пары (X, -X) для симметричных распределений.

Метод стратифицированной выборки

Разделяем область на подобласти и выбираем из каждой пропорционально.

Метод контрольных переменных

Используем коррелированную величину с известным математическим ожиданием.

Оценка погрешности

Стандартная ошибка

\[SE = \frac{s}{\sqrt{n}}\]

Количество испытаний для заданной точности

Для достижения точности ε с вероятностью 1-α:

\[n \geq \left(\frac{z_{\alpha/2}s}{\varepsilon}\right)^2\]

где z_{\alpha/2} - квантиль стандартного нормального распределения.

Псевдослучайные числа

Генераторы случайных чисел

  • Линейный конгруэнтный генератор
  • Генератор Мерсенна Твистера
  • Квазислучайные последовательности (Соболь, Хальтон)

Проверка качества

  • Тесты на равномерность
  • Тесты на независимость
  • Спектральные тесты

Параллельные вычисления

Естественный параллелизм

Каждая реализация независима, что идеально для параллельных вычислений.

Сходимость параллельной версии

Если использовать p процессоров по n/p испытаний каждый:

\[\hat{\mu}_{n}^{(p)} = \frac{1}{p}\sum_{j=1}^{p}\hat{\mu}_{n/p}^{(j)}\]

Сходимость такая же, как в последовательном случае.

Связь с другими теоремами

Закон больших чисел

Основа сходимости Монте-Карло оценок.

Центральная предельная теорема

Определяет скорость сходимости и позволяет строить доверительные интервалы.

Неравенство Чебышёва

Используется для доказательства сходимости по вероятности.

Историческая справка

Метод разработан в 1940-х годах в Лос-Аламосе (Джон фон Нейман, Станислав Улам).

Название связано с казино Монте-Карло.

Первоначально использовался для нейтронных расчётов в Манхэттенском проекте.
