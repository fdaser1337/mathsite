Теорема Тонелли

Формулировка

Для неотрицательной измеримой функции на произведении пространств повторные интегралы равны интегралу по произведению.

Условия

f ≥ 0 и измерима.

Идея доказательства

Доказательство через простые функции и монотонную сходимость.

Пример

Для индикатора квадрата [0,1]^2 все интегралы равны 1.
