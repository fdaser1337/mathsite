Теорема о сходимости итерационных методов

Условия сходимости итерационных методов решения систем линейных алгебраических уравнений.

Общая формулировка

Рассмотрим систему Ax = b, где A - невырожденная n×n матрица. Итерационный метод имеет вид:

\[x^{(k+1)} = Tx^{(k)} + c\]

где T - матрица перехода, c - вектор. Метод сходится к решению x* = A⁻¹b тогда и только тогда, когда:

\[\rho(T) < 1\]

где ρ(T) - спектральный радиус матрицы T (максимальное по модулю собственное значение).

Метод Якоби

Формула

Разложение A = D + L + U, где D - диагональная, L - нижнетреугольная, U - верхнетреугольная части.

\[x^{(k+1)} = D^{-1}(b - (L+U)x^{(k)})\]

Матрица перехода

\[T_J = -D^{-1}(L+U)\]

Условие сходимости

Метод Якоби сходится, если A строго или неприводимо диагонально доминирует:

\[|a_{ii}| > \sum_{j \neq i} |a_{ij}| \quad \text{для всех } i\]

Метод Гаусса-Зейделя

Формула

\[x^{(k+1)} = (D+L)^{-1}(b - Ux^{(k)})\]

Матрица перехода

\[T_{GS} = -(D+L)^{-1}U\]

Условие сходимости

Если A симметрическая положительно определённая, метод Гаусса-Зейделя всегда сходится.

Если A строго диагонально доминирует, метод сходится.

Метод сопряжённых градиентов

Формула

Для симметрической положительно определённой матрицы A:

\[r^{(k)} = b - Ax^{(k)}\] \[\alpha_k = \frac{(r^{(k)})^T r^{(k)}}{(r^{(k)})^T A r^{(k)}}\] \[x^{(k+1)} = x^{(k)} + \alpha_k r^{(k)}\] \[\beta_k = \frac{(r^{(k+1)})^T r^{(k+1)}}{(r^{(k)})^T r^{(k)}}\] \[r^{(k+1)} = r^{(k)} - \alpha_k A r^{(k)} + \beta_k r^{(k)}\]

Сходимость

Метод сходится не более чем за n итераций (в точной арифметике).

\[\|x^{(k)} - x^*\|_A \leq 2\left(\frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1}\right)^k \|x^{(0)} - x^*\|_A\]

где κ = cond₂(A) - число обусловленности.

Доказательство сходимости

Необходимое условие

Если метод сходится для любого начального приближения, то ρ(T) < 1.

Доказательство: если x* - решение, то x* = Tx* + c.

Тогда e^{(k)} = x^{(k)} - x* удовлетворяет e^{(k+1)} = Te^{(k)}.

Следовательно, e^{(k)} = T^k e^{(0)} → 0 для любого e^{(0)} только если ρ(T) < 1.

Достаточное условие

Если ρ(T) < 1, то метод сходится для любого начального приближения.

Используем норму, индуцированную спектральным радиусом.

Скорость сходимости

Асимптотическая скорость

\[R = -\ln \rho(T)\]

Сравнение методов

  • Якоби: медленная сходимость для плохо обусловленных матриц
  • Гаусса-Зейделя: обычно быстрее Якоби
  • Сопряжённые градиенты: оптимальный для SPD матриц

Примеры

Пример 1: Диагонально доминирующая матрица

A = [[4,1,1],[1,4,1],[1,1,4]], b = [6,6,6]ᵀ

Все три метода сходятся, метод Гаусса-Зейделя - быстрее всего.

Пример 2: Симметрическая положительно определённая матрица

A = [[2,-1,0],[-1,2,-1],[0,-1,2]]

Метод Гаусса-Зейделя и сопряжённых градиентов сходятся.

Пример 3: Расходящийся случай

A = [[1,2],[2,1]] - не диагонально доминирует, метод Якоби расходится.

Предобуславливание

Идея

Решаем M⁻¹Ax = M⁻¹b, где M ≈ A, но M⁻¹ легко вычисляется.

Методы предобуславливания

  • Якоби: M = diag(A)
  • Гаусса-Зейдель: M = D+L
  • SSOR: M = (D+ωL)D⁻¹(D+ωU)/ω(2-ω)
  • Неполное разложение Холецкого

Эффект

Хорошее предобуславливание уменьшает число обусловленности и ускоряет сходимость.

Практические аспекты

Критерии остановки

  • ‖b - Ax^{(k)}‖ < ε (невязка)
  • ‖x^{(k)} - x^{(k-1)}‖ < ε (изменение решения)
  • Относительная невязка < ε
  • Максимальное число итераций

Выбор метода

  • Разреженные матрицы: итерационные методы предпочтительнее
  • Плотные матрицы: прямые методы (LU-разложение)
  • SPD матрицы: метод сопряжённых градиентов
  • Общие матрицы: GMRES, BiCGSTAB

Обобщения

Нестационарные методы

GMRES, BiCGSTAB, QMR - меняют матрицу перехода на каждой итерации.

Многосеточные методы

Использование иерархии сеток для ускорения сходимости.

Доменно-декомпозиционные методы

Разделение области на подобласти с параллельной обработкой.
