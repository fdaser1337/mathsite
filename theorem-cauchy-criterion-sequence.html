Критерий Коши для сходимости последовательности

Формулировка

Последовательность x_n сходится тогда и только тогда, когда для любого ε > 0 существует N: при m,n ≥ N выполнено |x_n - x_m| < ε.

Условия

Работаем в полном пространстве R или C.

Идея доказательства

Необходимость следует из неравенства треугольника, достаточность — из полноты.

Пример

x_n = sum(k=1..n) 2^-k удовлетворяет критерию Коши, значит сходится.
