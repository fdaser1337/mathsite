# MathHub — сайт по базовому курсу высшей математики

Статический адаптивный сайт для студентов с ключевыми темами базового вышмата:

- математический анализ;
- линейная алгебра;
- аналитическая геометрия;
- ряды и последовательности;
- дифференциальные уравнения;
- теория вероятностей и статистика.

## Что уже реализовано

- структурированная подача по разделам;
- аккуратный современный дизайн;
- адаптивная вёрстка для мобильных и десктопов;
- переключение светлой/тёмной темы с сохранением в `localStorage`.

## Запуск

Откройте `index.html` в браузере или поднимите простой сервер:

```bash
python3 -m http.server 4173
```

После этого перейдите на `http://localhost:4173`.
