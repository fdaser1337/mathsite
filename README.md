# mathsite

Многостраничный учебный сайт по математике.

## Что есть сейчас
- Главная страница с блоком «Популярное».
- Отдельные объёмные разделы: алгебра, анализ, геометрия, теория вероятностей, теория чисел.
- Для каждой теоремы выделена отдельная страница.
- Формулы отображаются через MathJax (корректная поддержка записи вида `\(\lim_{x \to a} f(x)=L\)`).
