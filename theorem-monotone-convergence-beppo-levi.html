Теория меры

Теорема о монотонной сходимости

Для монотонно возрастающей последовательности измеримых функций интеграл предела равен пределу интегралов.

Формулировка

Пусть (X, Σ, μ) - пространство с мерой, {fₙ} - последовательность измеримых функций такая, что:

  1. 0 ≤ f₁(x) ≤ f₂(x) ≤ ... ≤ fₙ(x) ≤ ... для всех x ∈ X
  2. fₙ(x) → f(x) поточечно при n → ∞

Тогда:

\[\lim_{n \to \infty} \int_X f_n d\mu = \int_X f d\mu\]

При этом интеграл от f может быть бесконечным.

Интуитивное объяснение

Если последовательность функций монотонно возрастает к предельной функции, то «площадь под графиком» также монотонно возрастает к площади под графиком предела.

Аналогия: Если вы строите башню, добавляя этаж за этажом, то высота готовой башни равна сумме высот всех этажей.

Примеры

Пример 1: Ступенчатые функции

Пусть X = [0,1], μ - мера Лебега. Определим:

\[f_n(x) = \begin{cases} n, & x \in [0, 1/n] \\ 0, & x \in (1/n, 1] \end{cases}\]

Тогда fₙ ↑ 0, ∫fₙ = 1 для всех n, но ∫0 = 0. Теорема не применима, так как последовательность не монотонно возрастает.

Пример 2: Возрастающие функции

Пусть fₙ(x) = min(x, n) на [0,∞). Тогда fₙ ↑ f(x) = x.

\[\int_0^\infty f_n dx = n^2/2 \to \infty = \int_0^\infty f dx\]

Применения

1. Теория вероятностей

Предельные теоремы, математическое ожидание.

2. Функциональный анализ

L¹-пространства, теория операторов.

3. Гармонический анализ

Преобразования Фурье, свёртки.
