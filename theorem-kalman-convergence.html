Теорема о сходимости фильтра Калмана

Оптимальная линейная фильтрация и сходимость рекуррентных оценок.

Основная модель фильтра Калмана

Линейная стохастическая система

\[x_{k+1} = A_k x_k + B_k u_k + w_k\] \[z_k = H_k x_k + v_k\]

где:

  • x_k ∈ ℝ^n - вектор состояния
  • z_k ∈ ℝ^m - вектор наблюдений
  • u_k ∈ ℝ^p - вектор управления
  • w_k ~ N(0, Q_k) - шум процесса
  • v_k ~ N(0, R_k) - шум наблюдений

Предположения

  • w_k и v_k - независимые гауссовские шумы
  • Начальное состояние x₀ ~ N(μ₀, P₀)
  • Все матрицы известны

Алгоритм фильтра Калмана

Шаг предсказания

\[\hat{x}_{k|k-1} = A_k \hat{x}_{k-1|k-1} + B_k u_k\] \[P_{k|k-1} = A_k P_{k-1|k-1} A_k^T + Q_k\]

где \hat{x}_{k|k-1} - предсказанное состояние, P_{k|k-1} - ковариация ошибки предсказания.

Шаг коррекции

\[K_k = P_{k|k-1} H_k^T (H_k P_{k|k-1} H_k^T + R_k)^{-1}\] \[\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k (z_k - H_k \hat{x}_{k|k-1})\] \[P_{k|k} = (I - K_k H_k) P_{k|k-1}\]

где K_k - матрица усиления Калмана.

Теорема об оптимальности фильтра Калмана

Формулировка

Фильтр Калмана даёт оптимальную в среднеквадратичном смысле оценку состояния линейной гауссовой системы.

Свойства оптимальности

  • Несмещённость: E[x_k - \hat{x}_{k|k}] = 0
  • Минимальная дисперсия: tr(P_{k|k}) минимальна
  • Линейность: оценка - линейная функция наблюдений

Доказательство

Основано на ортогональности инноваций и проекции на подпространство наблюдений.

Теорема о сходимости стационарного фильтра

Формулировка

Для стационарной системы (A, H, Q, R постоянны) и при выполнении условий управляемости и наблюдаемости:

\[\lim_{k \to \infty} P_{k|k} = P_{\infty}\]

где P_{\infty} - решение алгебраического уравнения Риккати.

Алгебраическое уравнение Риккати

\[P_{\infty} = A P_{\infty} A^T - A P_{\infty} H^T (H P_{\infty} H^T + R)^{-1} H P_{\infty} A^T + Q\]

Условия сходимости

  • Пара (A, Q^{1/2}) управляема
  • Пара (A, H) наблюдаема
  • Собственные значения A лежат внутри единичной окружности

Теорема об устойчивости фильтра

Формулировка

Матрица усиления Калмана K_{\infty} обеспечивает асимптотическую устойчивость фильтра:

\[A_{\infty} = A - K_{\infty} H\]

имеет собственные значения внутри единичной окружности.

Следствие

\[\lim_{k \to \infty} (x_k - \hat{x}_{k|k}) = 0\]

в среднеквадратичном смысле.

Скорость сходимости

Экспоненциальная сходимость

\[\|P_{k|k} - P_{\infty}\| \leq C \lambda^k\]

где 0 < λ < 1 зависит от собственных значений A_{\infty}.

Факторы, влияющие на сходимость

  • Отношение сигнал/шум
  • Управляемость и наблюдаемость системы
  • Начальные условия P₀

Расширенный фильтр Калмана (EKF)

Линеаризация

Для нелинейной системы:

\[x_{k+1} = f(x_k, u_k) + w_k\] \[z_k = h(x_k) + v_k\]

Линеаризуем вокруг текущей оценки:

\[A_k = \frac{\partial f}{\partial x}\Big|_{\hat{x}_{k|k}}, \quad H_k = \frac{\partial h}{\partial x}\Big|_{\hat{x}_{k|k-1}}\]

Сходимость EKF

Локальная сходимость гарантируется при малых ошибках линеаризации.

Фильтр Калмана-Бьюси

Сглаживание

Оценка состояния x_k по всем наблюдениям до времени t > k:

\[\hat{x}_{k|t} = \hat{x}_{k|k} + C_k (\hat{x}_{k+1|t} - \hat{x}_{k+1|k})\]

Матрица сглаживания

\[C_k = P_{k|k} A_{k+1}^T P_{k+1|k}^{-1}\]

Теорема о декомпозиции фильтра

UD-декомпозиция

Численно устойчивая реализация:

\[P_{k|k-1} = U D U^T\]

где U - унитриангулярная, D - диагональная матрицы.

Преимущества

  • Гарантирует положительную определённость P
  • Улучшенная численная устойчивость
  • Снижение вычислительных затрат

Адаптивный фильтр Калмана

Оценка ковариаций

Адаптивная оценка Q_k и R_k по инновациям:

\[r_k = z_k - H_k \hat{x}_{k|k-1}\]

Методы адаптации

  • Метод максимального правдоподобия
  • Байесовские методы
  • Методы инновационного анализа

Практические аспекты

Численные проблемы

  • Численная неустойчивость при плохой обусловленности
  • Дивергенция при неверных моделях
  • Выбор начальных условий

Вычислительная сложность

  • Стандартный фильтр: O(n³ + n²m + nm²)
  • Разреженные матрицы: O(n²)
  • UD-фильтр: O(n²)
