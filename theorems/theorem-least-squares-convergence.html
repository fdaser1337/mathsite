Теорема о сходимости метода наименьших квадратов

Состоятельность и асимптотическая нормальность оценок метода наименьших квадратов.

Основная постановка

Рассмотрим линейную регрессионную модель:

\[y_i = x_i^T \beta + \varepsilon_i, \quad i = 1, 2, ..., n\]

где:

  • y_i - наблюдаемые значения зависимой переменной
  • x_i ∈ ℝ^p - вектор объясняющих переменных
  • β ∈ ℝ^p - вектор неизвестных параметров
  • ε_i - случайные ошибки

В матричной форме: y = Xβ + ε, где X ∈ ℝ^{n×p} - матрица регрессоров.

Оценка метода наименьших квадратов

Формула оценки

Оценка МНК минимизирует сумму квадратов остатков:

\[\hat{\beta} = \arg\min_{\beta} \sum_{i=1}^{n} (y_i - x_i^T \beta)^2\]

Нормальные уравнения

Дифференцируя по β и приравнивая к нулю, получаем:

\[X^T X \hat{\beta} = X^T y\]

Решение

Если X^T X обратима, то:

\[\hat{\beta} = (X^T X)^{-1} X^T y\]

Теорема Гаусса-Маркова

Формулировка

Пусть выполнены условия:

  1. E[ε] = 0 (несмещённость ошибок)
  2. Cov(ε) = σ²I_n (гомоскедастичность и некоррелированность)
  3. rank(X) = p (полный ранг матрицы регрессоров)

Тогда оценка МНК \hat{β} является наилучшей линейной несмещённой оценкой (BLUE).

Свойства оценки

  • Несмещённость: E[\hat{β}] = β
  • Ковариационная матрица: Cov(\hat{β}) = σ²(X^T X)^{-1}
  • Эффективность: минимальная дисперсия среди линейных несмещённых оценок

Сходимость при n → ∞

Теорема состоятельности

Пусть выполнены условия:

  1. E[ε_i] = 0, Var(ε_i) = σ² < ∞
  2. \lim_{n→∞} (1/n)X^T X = Q, где Q - положительно определённая матрица
  3. \max_{i≤n} x_i^T(X^T X)^{-1}x_i → 0

Тогда оценка МНК состоятельна:

\[\hat{\beta}_n \xrightarrow{P} \beta\]

Доказательство

Запишем оценку в виде:

\[\hat{\beta} = \beta + (X^T X)^{-1} X^T \varepsilon\]

Поскольку (X^T X)^{-1}X^T ε → 0 по вероятности, получаем состоятельность.

Асимптотическая нормальность

Теорема

При дополнительных условиях регулярности:

\[\sqrt{n}(\hat{\beta}_n - \beta) \xrightarrow{d} \mathcal{N}(0, \sigma^2 Q^{-1})\]

Следствия

  • Асимптотическое распределение оценки нормально
  • Можно строить асимптотические доверительные интервалы
  • Проверка гипотез с помощью асимптотических тестов

Примеры

Пример 1: Простая линейная регрессия

y_i = β₀ + β₁x_i + ε_i

Оценки:

\[\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}\] \[\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}\]

Пример 2: Полиномиальная регрессия

y_i = β₀ + β₁x_i + β₂x_i² + ε_i

Матрица X содержит столбцы [1, x_i, x_i²].

Пример 3: Множественная регрессия

y_i = β₀ + β₁x_{i1} + β₂x_{i2} + ε_i

Оценки получаются из нормальных уравнений.

Обобщения метода наименьших квадратов

Взвешенный МНК

Если Var(ε_i) = σ²/w_i, то:

\[\hat{\beta}_W = (X^T W X)^{-1} X^T W y\]

где W = diag(w₁, ..., wₙ).

Обобщённый МНК

Для Cov(ε) = Σ:

\[\hat{\beta}_{GLS} = (X^T Σ^{-1} X)^{-1} X^T Σ^{-1} y\]

Ридж-регрессия

Добавляем L₂-регуляризацию:

\[\hat{\beta}_{ridge} = (X^T X + \lambda I)^{-1} X^T y\]

Численные аспекты

Проблема мультиколлинеарности

Если столбцы X почти линейно зависимы, то X^T X плохо обусловлена.

Методы решения

  • QR-разложение: X = QR, \hat{β} = R^{-1}Q^T y
  • SVD-разложение: X = UΣV^T, \hat{β} = VΣ^{-1}U^T y
  • Метод Холецкого для X^T X

Вычислительная сложность

  • Прямой метод: O(np²)
  • Итерационные методы: O(kp²), где k - число итераций

Диагностика модели

Коэффициент детерминации R²

\[R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}\]

Остатки и их свойства

Остатки: e_i = y_i - \hat{y}_i

При выполнении условий Гаусса-Маркова:

  • E[e_i] = 0
  • ∑e_i = 0
  • ∑x_{ij}e_i = 0 для всех j

Связь с другими теоремами

Теорема Пифагора

МНК проецирует y на подпространство, натянутое на столбцы X.

Теорема об ортогональном проецировании

Остатки ортогональны пространству регрессоров.

Центральная предельная теорема

Используется для доказательства асимптотической нормальности.

Практические применения

1. Эконометрика

Анализ временных рядов, модели спроса и предложения.

2. Машинное обучение

Линейная регрессия, полиномиальные признаки.

3. Инженерия

Аппроксимация экспериментальных данных.

4. Биостатистика

Анализ клинических исследований, эпидемиология.

Историческая справка

Метод разработан Карлом Гауссом (1809) и Адриеном Лежандром (1805).

Гаусс использовал метод для предсказания орбиты астероида Церера.
