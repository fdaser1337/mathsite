Теорема о сходимости разностных схем

Эквивалентность устойчивости и сходимости для линейных разностных схем.

Основная постановка

Рассмотрим линейную начально-краевую задачу:

\[\frac{\partial u}{\partial t} = \mathcal{L}u + f, \quad x \in \Omega, \quad t > 0\] \[Bu = g, \quad x \in \partial\Omega\] \[u(x,0) = u_0(x), \quad x \in \Omega\]

где ℒ - линейный дифференциальный оператор.

Разностная схема аппроксимирует эту задачу на сетке с шагом h по пространству и τ по времени.

Теорема Лакса

Формулировка

Для корректно поставленной линейной задачи и консистентной разностной схемы:

\[\text{Устойчивость} \iff \text{Сходимость}\]

Более точно: если разностная схема устойчива и аппроксимирует исходную задачу с порядком O(h^p + τ^q), то она сходится с тем же порядком.

Определения

  • Корректность: решение существует, единственно, непрерывно зависит от данных
  • Консистентность: погрешность аппроксимации → 0 при h,τ → 0
  • Устойчивость: решение разностной задачи ограничено равномерно по h,τ
  • Сходимость: разностное решение → точное решение

Доказательство теоремы Лакса

Основная идея

Пусть u - точное решение, u_h - разностное решение.

Разложим ошибку e_h = u_h - u:

\[e_h = (u_h - P_h u) + (P_h u - u)\]

где P_h - оператор проектирования на сетку.

Оценка ошибки

Используем треугольное неравенство:

\[\|e_h\| \leq \|u_h - P_h u\| + \|P_h u - u\|\]

Первый член оценивается через устойчивость, второй - через консистентность.

Примеры разностных схем

Пример 1: Уравнение теплопроводности

∂u/∂t = ∂²u/∂x², u(0,t) = u(1,t) = 0, u(x,0) = f(x).

Явная схема:

\[\frac{u_i^{n+1} - u_i^n}{\tau} = \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{h^2}\]

Условие устойчивости: τ ≤ h²/2 (условие Куранта).

Неявная схема:

\[\frac{u_i^{n+1} - u_i^n}{\tau} = \frac{u_{i+1}^{n+1} - 2u_i^{n+1} + u_{i-1}^{n+1}}{h^2}\]

Безусловно устойчива.

Метод Фурье анализа устойчивости

Идея

Для линейных схем с постоянными коэффициентами ищем решение в виде:

\[u_i^n = \xi^n e^{ikih}\]

Коэффициент усиления

Подставляем в разностную схему, получаем:

\[\xi = G(kh, \tau/h)\]

Условие фон Неймана

Схема устойчива, если |ξ| ≤ 1 + O(τ) для всех k.

Для явной схемы теплопроводности:

\[\xi = 1 - 4\frac{\tau}{h^2}\sin^2\left(\frac{kh}{2}\right)\]

Условие |ξ| ≤ 1 даёт τ ≤ h²/2.

Теорема Лакса-Рихтмайера

Обобщение на эволюционные задачи

Для начальной задачи u' = Au, где A - оператор, разностная схема:

\[u^{n+1} = R(\tau A)u^n + \tau Q(\tau A)f^n\]

Схема сходится, если:

  1. ‖R(τA)‖ ≤ C для τA в области устойчивости
  2. R(0) = I, Q(0) = I (консистентность)

Порядок аппроксимации

Определение

Схема имеет порядок (p,q), если погрешность аппроксимации:

\[\|\tau_h\| = O(h^p + \tau^q)\]

Примеры порядков

  • Явная схема теплопроводности: O(τ + h²)
  • Неявная схема теплопроводности: O(τ + h²)
  • Схема Кранка-Николсон: O(τ² + h²)
  • Схема Лакса-Вендроффа: O(τ² + h²)

Метод энергетических оценок

Идея

Используем энергетические неравенства для доказательства устойчивости.

Пример для волнового уравнения

u_tt = c²u_xx, разностная схема:

\[\frac{u_i^{n+1} - 2u_i^n + u_i^{n-1}}{\tau^2} = c^2\frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{h^2}\]

Дискретная энергия:

\[E^n = h\sum_i \left(\frac{u_i^{n+1} - u_i^{n-1}}{2\tau}\right)^2 + c^2 h\sum_i \left(\frac{u_{i+1}^n - u_{i-1}^n}{2h}\right)^2\]

Условие устойчивости: τ ≤ h/c.

Нелинейные схемы

Сложности

Для нелинейных задач теорема Лакса не применима напрямую.

Линеаризация

Многомерные задачи

Схемы расщепления

Аддитивные схемы

Суммирование одномерных операторов.

Условия устойчивости

Более строгие, чем в одномерном случае.

Практические аспекты

Выбор шага

  • Явные схемы: ограничение по условию Куранта
  • Неявные схемы: неограничены по шагу, но требуют решения СЛАУ
  • Компромисс: неявно-явные схемы

Численная реализация

  • Эффективные алгоритмы решения СЛАУ
  • Параллельные вычисления
  • Адаптивные сетки
