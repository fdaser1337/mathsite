Теорема о сходимости HMM

Сходимость скрытых марковских моделей и алгоритма Баума-Велша.

Основная модель HMM

Скрытая марковская модель определяется параметрами λ = (A, B, π):

Компоненты модели

  • Скрытые состояния: S = {s₁, ..., s_N}
  • Наблюдения: O = {o₁, ..., o_M}
  • Матрица переходов: A = [a_{ij}], где a_{ij} = P(q_{t+1} = s_j|q_t = s_i)
  • Матрица эмиссий: B = [b_j(k)], где b_j(k) = P(o_t = o_k|q_t = s_j)
  • Начальное распределение: π = [π_i], где π_i = P(q₁ = s_i)

Условия

  • ∑_j a_{ij} = 1 для всех i
  • ∑_k b_j(k) = 1 для всех j
  • ∑_i π_i = 1

Основные задачи HMM

Задача 1: Оценка вероятности

Дана последовательность наблюдений O = o₁, ..., o_T и модель λ. Найти P(O|λ).

Задача 2: Декодирование

Даны O и λ. Найти наиболее вероятную последовательность состояний Q = q₁, ..., q_T.

Задача 3: Обучение

Даны O. Найти параметры λ, максимизирующие P(O|λ).

Прямой и обратный алгоритмы

Прямая переменная

\[\alpha_t(i) = P(o_1, ..., o_t, q_t = s_i|\lambda)\]

Инициализация

\[\alpha_1(i) = \pi_i b_i(o_1)\]

Рекурсия

\[\alpha_{t+1}(j) = \left[\sum_{i=1}^{N} \alpha_t(i) a_{ij}\right] b_j(o_{t+1})\]

Завершение

\[P(O|\lambda) = \sum_{i=1}^{N} \alpha_T(i)\]

Обратная переменная

\[\beta_t(i) = P(o_{t+1}, ..., o_T|q_t = s_i, \lambda)\]

Алгоритм Баума-Велша (EM для HMM)

E-шаг: вычисление ожиданий

\[\gamma_t(i) = P(q_t = s_i|O, \lambda) = \frac{\alpha_t(i)\beta_t(i)}{\sum_{j=1}^{N} \alpha_t(j)\beta_t(j)}\]
\[\xi_t(i,j) = P(q_t = s_i, q_{t+1} = s_j|O, \lambda) = \frac{\alpha_t(i) a_{ij} b_j(o_{t+1}) \beta_{t+1}(j)}{\sum_{k,l=1}^{N} \alpha_t(k) a_{kl} b_l(o_{t+1}) \beta_{t+1}(l)}\]

M-шаг: обновление параметров

\[\bar{\pi}_i = \gamma_1(i)\]
\[\bar{a}_{ij} = \frac{\sum_{t=1}^{T-1} \xi_t(i,j)}{\sum_{t=1}^{T-1} \gamma_t(i)}\]
\[\bar{b}_j(k) = \frac{\sum_{t=1, o_t = o_k}^{T} \gamma_t(j)}{\sum_{t=1}^{T} \gamma_t(j)}\]

Теорема о сходимости алгоритма Баума-Велша

Формулировка

Алгоритм Баума-Велша монотонно увеличивает правдоподобие P(O|λ) и сходится к локальному максимуму.

Свойства сходимости

  • P(O|λ^{(t+1)}) ≥ P(O|λ^{(t)}) (монотонность)
  • Сходимость к стационарной точке правдоподобия
  • Локальная, а не глобальная оптимальность

Условия сходимости

  • Наблюдения достаточно информативны
  • Модель идентифицируема
  • Начальные параметры не вырождены

Теорема об идентифицируемости HMM

Формулировка

HMM идентифицируема (с точностью до переименования состояний) если:

  1. Матрицы A и B неразложимы
  2. Наблюдения достаточно длинные и разнообразные
  3. Начальное распределение π имеет все положительные компоненты

Неоднозначность

  • Перенумерация состояний не меняет модель
  • Возможны эквивалентные представления

Теорема о состоятельности оценок

Формулировка

При T → ∞ оценки параметров HMM сходятся к истинным значениям с вероятностью 1.

Скорость сходимости

\[\|\hat{\lambda}_T - \lambda_0\| = O_p\left(\frac{1}{\sqrt{T}}\right)\]

Условия

  • Эргодичность марковской цепи
  • Идентифицируемость модели
  • Регулярность матриц переходов и эмиссий

Алгоритм Витерби

Задача декодирования

Нахождение наиболее вероятной последовательности состояний:

\[Q^* = \arg\max_Q P(Q|O, \lambda)\]

Динамическое программирование

\[\delta_t(i) = \max_{q_1, ..., q_{t-1}} P(q_1, ..., q_{t-1}, q_t = s_i, o_1, ..., o_t|\lambda)\]

Рекурсия

\[\delta_{t+1}(j) = \max_i [\delta_t(i) a_{ij}] b_j(o_{t+1})\]

Восстановление пути

\[\psi_t(j) = \arg\max_i [\delta_{t-1}(i) a_{ij}]\]

Теорема о сложности HMM

Вычислительная сложность

  • Прямой/обратный алгоритм: O(N²T)
  • Алгоритм Баума-Велша: O(N²T) на итерацию
  • Алгоритм Витерби: O(N²T)

Память

  • Прямой алгоритм: O(NT)
  • Обратный алгоритм: O(NT)
  • Витерби: O(NT)

Расширения HMM

HMM с непрерывными наблюдениями

Использование гауссовых смесей для эмиссионных вероятностей.

Скрытые полумарковские модели (HSMM)

Учёт длительности пребывания в состояниях.

Иерархические HMM

Многоуровневые модели с иерархией состояний.

Факторные HMM

Независимые цепи с общими наблюдениями.

Практические аспекты

Проблемы

  • Локальные оптимумы в обучении
  • Численная нестабильность (underflow)
  • Выбор числа состояний

Решения

  • Множественные инициализации
  • Логарифмическая шкала вычислений
  • Критерии AIC/BIC для выбора модели

Регуляризация

  • Сглаживание оценок (аддитивное сглаживание)
  • Байесовские HMM с априорными распределениями

Связь с другими методами

Марковские цепи

HMM - обобщение марковских цепей с неполными наблюдениями.

EM-алгоритм

Баум-Велш - частный случай EM для дискретных скрытых переменных.

Фильтр Калмана

Линейный гауссовский аналог HMM.

Применения

1. Распознавание речи

Моделирование фонем и слов.

2. Биоинформатика

Анализ последовательностей ДНК, поиск генов.

3. Финансы

Моделирование рыночных режимов.

4. Обработка сигналов

Анализ временных рядов, детекция аномалий.

Историческая справка

HMM предложены Леонардом Баумом и коллегами в 1960-х годах.

Алгоритм Баума-Велша разработан в 1970-х годах.

Широкое применение началось с 1980-х годов.
