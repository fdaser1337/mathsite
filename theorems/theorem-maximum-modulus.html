Принцип максимума модуля

Модуль аналитической функции достигает своего максимума на границе области.

Формулировка

Пусть f - непостоянная аналитическая функция в ограниченной области D с непрерывным продолжением на границу ∂D. Тогда:

\[\max_{z \in \overline{D}} |f(z)| = \max_{z \in \partial D} |f(z)|\]

Более того, если |f(z₀)| = max|f(z)| для некоторой внутренней точки z₀ ∈ D, то f ≡ const.

Геометрическая интерпретация

График модуля аналитической функции похож на «гору» - самая высокая точка всегда на границе области.

Аналогия: Если вы находитесь в долине и поднимаетесь в гору, то самая высокая вершина всегда будет на границе долины, а не внутри.

Доказательство

Используем принцип максимума для гармонических функций.

Пусть u(z) = |f(z)|² = f(z)\overline{f(z)}. Если f(z) ≠ 0, то:

\[\Delta u = 4|f'(z)|^2 \geq 0\]

Таким образом, u - субгармоническая функция. По принципу максимума для субгармонических функций, максимум достигается на границе.

Если максимум достигается внутри, то u ≡ const, следовательно, f'(z) ≡ 0, и f ≡ const.

Примеры

Пример 1: Константа

f(z) = 5. Модуль постоянен, принцип выполняется тривиально.

Пример 2: Линейная функция

f(z) = z в единичном круге |z| ≤ 1.

\[\max_{|z| \leq 1} |z| = 1 = \max_{|z| = 1} |z|\]

Пример 3: Квадратичная функция

f(z) = z² в круге |z| ≤ 2.

\[\max_{|z| \leq 2} |z^2| = 4 = \max_{|z| = 2} |z^2|\]

Пример 4: Экспонента

f(z) = eᶻ в прямоугольнике 0 ≤ Re(z) ≤ 1, 0 ≤ Im(z) ≤ π.

\[\max |e^z| = e^1\]

Следствия

1. Принцип минимума модуля

Если f(z) ≠ 0 в D, то |f(z)| достигает минимума на границе.

2. Лемма Шварца

Если f аналитична в единичном круге, |f(z)| ≤ 1 и f(0) = 0, то |f(z)| ≤ |z|.

3. Теорема о среднем значении

Значение аналитической функции в центре круга равно среднему значению на границе.

Обобщения
