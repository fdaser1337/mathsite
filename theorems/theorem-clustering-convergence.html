Теорема о сходимости кластеризации

Сходимость алгоритмов кластеризации и свойства оптимальных разбиений.

k-means алгоритм

Целевая функция

Минимизация внутрикластерной дисперсии:

\[J(C, \mu) = \sum_{k=1}^{K}\sum_{x_i \in C_k} \|x_i - \mu_k\|^2\]

где C_k - кластер k, μ_k - центр кластера k.

Итерационный алгоритм

  1. Шаг назначения: x_i ∈ C_{argmin_k}‖x_i - μ_k‖
  2. Шаг обновления: μ_k = (1/|C_k|)∑_{x_i∈C_k} x_i

Теорема о сходимости k-means

Формулировка

Алгоритм k-means всегда сходится за конечное число итераций к локальному минимуму.

Доказательство

  • Каждая итерация не увеличивает целевую функцию J
  • Существует конечное число возможных разбиений (n^K)
  • Алгоритм не может циклироваться без уменьшения J
  • Следовательно, достигается локальный минимум за конечное число шагов

Следствия

  • Сходимость гарантирована, но не к глобальному оптимуму
  • Результат зависит от начальной инициализации
  • Множественные запуски улучшают качество решения

EM-алгоритм для гауссовых смесей

Модель

Смесь гауссовских распределений:

\[p(x) = \sum_{k=1}^{K} \pi_k \mathcal{N}(x|\mu_k, \Sigma_k)\]

E-шаг

\[\gamma_{ik}^{(t)} = \frac{\pi_k^{(t)} \mathcal{N}(x_i|\mu_k^{(t)}, \Sigma_k^{(t)})}{\sum_{j=1}^{K} \pi_j^{(t)} \mathcal{N}(x_i|\mu_j^{(t)}, \Sigma_j^{(t)})}\]

M-шаг

\[\pi_k^{(t+1)} = \frac{1}{n}\sum_{i=1}^{n} \gamma_{ik}^{(t)}\] \[\mu_k^{(t+1)} = \frac{\sum_{i=1}^{n} \gamma_{ik}^{(t)} x_i}{\sum_{i=1}^{n} \gamma_{ik}^{(t)}}\] \[\Sigma_k^{(t+1)} = \frac{\sum_{i=1}^{n} \gamma_{ik}^{(t)}(x_i - \mu_k^{(t+1)})(x_i - \mu_k^{(t+1)})^T}{\sum_{i=1}^{n} \gamma_{ik}^{(t)}}\]

Теорема о сходимости EM-алгоритма

Формулировка

EM-алгоритм монотонно увеличивает правдоподобие и сходится к стационарной точке.

Свойства сходимости

  • ℓ(θ^{(t+1)}) ≥ ℓ(θ^{(t)}) (монотонность правдоподобия)
  • Сходимость к локальному максимуму правдоподобия
  • Линейная скорость сходимости вблизи оптимума

Условия сходимости

  • Правдоподобие ограничено сверху
  • Полная вероятность положительна
  • Регулярность модели

Теорема о консистентности

Формулировка

При n → ∞ оценки параметров смеси сходятся к истинным значениям.

Условия

  • Идентифицируемость компонентов смеси
  • Разделённость компонентов
  • Правильное число кластеров K

Скорость сходимости

\[\|\hat{\theta}_n - \theta_0\| = O_p(n^{-1/2})\]

Иерархическая кластеризация

Агломеративный подход

Последовательное объединение ближайших кластеров.

Критерии связывания

  • Single linkage: min_{x∈C_i, y∈C_j} d(x,y)
  • Complete linkage: max_{x∈C_i, y∈C_j} d(x,y)
  • Average linkage: (1/|C_i||C_j|)∑_{x∈C_i, y∈C_j} d(x,y)
  • Ward's method: минимизация прироста дисперсии

Свойства

  • Детерминированный алгоритм
  • Дендрограмма представления иерархии
  • Различные свойства связывания

Теорема об устойчивости кластеризации

Определение устойчивости

Кластеризация устойчива, если малые изменения данных не меняют структуру кластеров.

Теорема

Для хорошо разделённых кластеров с минимальным расстоянием δ:

\[P(\text{изменение кластеризации}) \leq \exp\left(-\frac{n\delta^2}{8\sigma^2}\right)\]

где σ² - максимальная дисперсия внутри кластеров.

Спектральная кластеризация

Идея

Использование собственных векторов матрицы сходства для кластеризации.

Алгоритм

  1. Построение матрицы сходства W
  2. Вычисление нормализованной лапласиана L = D^{-1/2}WD^{-1/2}
  3. Нахождение K первых собственных векторов
  4. k-means в пространстве собственных векторов

Теорема о сходимости

При определённых условиях спектральная кластеризация сходится к оптимальному разбиению.

Оценка качества кластеризации

Silhouette коэффициент

\[s(i) = \frac{b(i) - a(i)}{\max\{a(i), b(i)\}}\]

где a(i) - среднее расстояние до точек того же кластера, b(i) - минимальное среднее расстояние до точек других кластеров.

Индекс Данна

\[DB = \frac{1}{K}\sum_{k=1}^{K} \max_{j \neq k} \frac{S_k + S_j}{d(C_k, C_j)}\]

Теорема о числе кластеров

Метод локтя

Выбор K по точке изгиба графика J(K).

Информационные критерии

\[AIC = -2\log L + 2p\] \[BIC = -2\log L + p\log n\]

где p - число параметров, L - правдоподобие.

Практические аспекты

Выбор алгоритма

  • k-means: сферические кластеры, большие данные
  • EM-алгоритм: гауссовы смеси, вероятностная кластеризация
  • Иерархическая:探索ная структура данных
  • Спектральная: сложные формы кластеров

Масштабирование

  • k-means: O(nKI) - линейно по числу точек
  • EM: O(nKI) - линейно по числу точек
  • Иерархическая: O(n² log n) - квадратично
  • Спектральная: O(n³) - кубично
