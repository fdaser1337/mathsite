Теорема о сходимости метода Ньютона

Условия сходимости и скорость сходимости итерационного метода Ньютона для решения уравнений.

Формулировка

Пусть f: [a,b] → ℝ - дважды непрерывно дифференцируемая функция. Предположим:

  1. Существует корень α ∈ [a,b] такой, что f(α) = 0
  2. f'(x) ≠ 0 для всех x ∈ [a,b]
  3. Начальное приближение x₀ достаточно близко к α
  4. Выполнено условие |f(x)f''(x)|/f'(x)² ≤ M < 1 в окрестности α

Тогда итерационная последовательность:

\[x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\]

сходится к α и сходимость квадратичная:

\[|x_{n+1} - \alpha| \leq C|x_n - \alpha|^2\]

Квадратичная сходимость

Ключевое свойство метода Ньютона - квадратичная сходимость:

\[\lim_{n \to \infty} \frac{|x_{n+1} - \alpha|}{|x_n - \alpha|^2} = \frac{|f''(\alpha)|}{2|f'(\alpha)|}\]

Это означает, что на каждой итерации количество верных цифр примерно удваивается.

Сравнение с другими методами:

  • Метод Ньютона: квадратичная сходимость
  • Метод секущих: порядок сходимости φ ≈ 1.618
  • Метод простой итерации: линейная сходимость

Достаточные условия сходимости

Условие Канторовича

Пусть выполнено:

\[\|f'(x_0)^{-1}f(x_0)\| \cdot h \leq \frac{1}{2}\]

где h - константа Липшица для f'(x). Тогда метод сходится.

Условие остойчивости

Если f'(α) ≠ 0 и f''(x) непрерывна в окрестности α, то существует δ > 0 такое, что для любого x₀ ∈ (α-δ, α+δ) метод сходится.

Доказательство сходимости

Разложение в ряд Тейлора

Используем разложение f(xₙ) в окрестности α:

\[f(x_n) = f(\alpha) + f'(\alpha)(x_n - \alpha) + \frac{f''(\xi_n)}{2}(x_n - \alpha)^2\]

Так как f(α) = 0, получаем:

\[x_{n+1} - \alpha = x_n - \alpha - \frac{f(x_n)}{f'(x_n)}\]

Оценка погрешности

После преобразований получаем:

\[|x_{n+1} - \alpha| \leq \frac{M}{2m}|x_n - \alpha|^2\]

где M = max|f''(x)|, m = min|f'(x)| в окрестности α.

Примеры

Пример 1: Вычисление квадратного корня

Для f(x) = x² - a, метод Ньютона даёт:

\[x_{n+1} = \frac{1}{2}\left(x_n + \frac{a}{x_n}\right)\]

Это классический метод вычисления √a с квадратичной сходимостью.

Пример 2: Решение x³ - 2 = 0

f(x) = x³ - 2, f'(x) = 3x². Итерации:

\[x_{n+1} = x_n - \frac{x_n^3 - 2}{3x_n^2} = \frac{2x_n^3 + 2}{3x_n^2}\]

Сходимость к ∛2 ≈ 1.259921.

Пример 3: Расходимость при плохом начальном приближении

Для f(x) = arctan(x), начальное x₀ = 10 приводит к расходимости.

Многомерный случай

Для системы F(x) = 0, где F: ℝⁿ → ℝⁿ:

\[x_{k+1} = x_k - J_F(x_k)^{-1}F(x_k)\]

где J_F(x) - матрица Якоби.

Условия сходимости:

  • J_F(α) - невырожденная матрица
  • F дважды непрерывно дифференцируема
  • Начальное приближение достаточно близко к решению

Модификации метода

Метод Ньютона с демпфированием

\[x_{n+1} = x_n - \lambda_n \frac{f(x_n)}{f'(x_n)}\]

где 0 < λₙ ≤ 1 - параметр демпфирования.

Квази-ньютоновские методы

Приближённое вычисление обратной матрицы Якоби.

Метод секущих

Замена производной на разностное отношение.

Практические аспекты

Критерий остановки

  • |x_{n+1} - xₙ| < ε
  • |f(xₙ)| < ε
  • Относительная погрешность < ε

Проблемы

  • Выбор начального приближения
  • Вычисление производной
  • Расходимость при f'(xₙ) ≈ 0

Области сходимости

Бассейны притяжения

Множество начальных точек, сходящихся к данному корню.

Границы бассейнов могут иметь фрактальную структуру.

Глобальная сходимость

Для выпуклых функций метод сходится из любой начальной точки.

Связь с другими методами

Метод простой итерации

x_{n+1} = xₙ - λf(xₙ) - линейная сходимость.

Метод градиентного спуска

Для минимизации функций - линейная сходимость.

Метод Ньютона для оптимизации

x_{n+1} = xₙ - H_f(xₙ)⁻¹∇f(xₙ) для минимизации f(x).

Историческая справка

Исаак Ньютон (1643-1727) предложил метод в «Методе флюксий».

Джозеф Рафсон (1648-1715) опубликовал современную формулировку.

Строгая теория сходимости развита в XIX-XX веках.
