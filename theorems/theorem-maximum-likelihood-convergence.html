Теорема о сходимости метода максимального правдоподобия

Состоятельность и асимптотическая нормальность оценок максимального правдоподобия.

Основная постановка

Пусть X₁, X₂, ..., Xₙ - независимые одинаково распределённые случайные величины с плотностью p(x|θ), где θ ∈ Θ ⊂ ℝ^k - неизвестный параметр.

Функция правдоподобия:

\[L_n(\theta) = \prod_{i=1}^{n} p(X_i|\theta)\]

Логарифмическая функция правдоподобия:

\[\ell_n(\theta) = \log L_n(\theta) = \sum_{i=1}^{n} \log p(X_i|\theta)\]

Оценка максимального правдоподобия:

\[\hat{\theta}_n = \arg\max_{\theta \in \Theta} L_n(\theta) = \arg\max_{\theta \in \Theta} \ell_n(\theta)\]

Условия регулярности

Для сходимости ММП-оценок требуются следующие условия:

  1. Идентифицируемость: если θ₁ ≠ θ₂, то p(x|θ₁) ≠ p(x|θ₂) на множестве положительной меры
  2. Компактность: Θ - компактное множество
  3. Непрерывность: p(x|θ) непрерывна по θ для почти всех x
  4. Интегрируемость: E[|log p(X|θ)|] < ∞ для всех θ ∈ Θ
  5. Дифференцируемость: p(x|θ) дважды дифференцируема по θ
  6. Неособенность информационной матрицы: I(θ₀) положительно определена

Теорема состоятельности

Формулировка

При выполнении условий регулярности оценка максимального правдоподобия состоятельна:

\[\hat{\theta}_n \xrightarrow{P} \theta_0\]

где θ₀ - истинное значение параметра.

Основная идея доказательства

Используем закон больших чисел:

\[\frac{1}{n}\ell_n(\theta) = \frac{1}{n}\sum_{i=1}^{n} \log p(X_i|\theta) \xrightarrow{P} E[\log p(X|\theta)] = \ell(\theta)\]

По неравенству Йенсена:

\[\ell(\theta) = E[\log p(X|\theta)] < \log E[p(X|\theta)] = \log p(X|\theta_0) = \ell(\theta_0)\]

для θ ≠ θ₀. Следовательно, максимум эмпирической функции сходится к максимуму теоретической.

Теорема асимптотической нормальности

Формулировка

При дополнительных условиях дифференцируемости:

\[\sqrt{n}(\hat{\theta}_n - \theta_0) \xrightarrow{d} \mathcal{N}(0, I^{-1}(\theta_0))\]

где I(θ) - информационная матрица Фишера:

\[I(\theta) = -E\left[\frac{\partial^2 \log p(X|\theta)}{\partial \theta \partial \theta^T}\right]\]

Доказательство

Разлагаем score-функцию в ряд Тейлора:

\[S_n(\theta) = \frac{\partial \ell_n(\theta)}{\partial \theta} = S_n(\theta_0) + H_n(\theta^*)(\theta - \theta_0)\]

где H_n(θ) - матрица вторых производных, θ* между θ и θ₀.

Так как S_n(\hat{θ}_n) = 0, получаем:

\[\hat{\theta}_n - \theta_0 = -H_n^{-1}(\theta^*) S_n(\theta_0)\]

По ЦПТ: S_n(θ₀)/√n → N(0, I(θ₀)), а H_n(θ*) → -I(θ₀).

Теорема эффективности

Формулировка

Оценка максимального правдоподобия асимптотически эффективна:

\[\lim_{n \to \infty} n \cdot \text{Var}(\hat{\theta}_n) = I^{-1}(\theta_0)\]

Это означает, что ММП-оценка достигает границы Крамера-Рао.

Следствие

Любая другая состоятельная оценка не может иметь меньшую асимптотическую дисперсию.

Примеры

Пример 1: Нормальное распределение

X₁, ..., Xₙ ~ N(μ, σ²), θ = (μ, σ²).

Логарифмическое правдоподобие:

\[\ell_n(\mu, \sigma^2) = -\frac{n}{2}\log(2\pi) - \frac{n}{2}\log(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(X_i - \mu)^2\]

ММП-оценки: \hat{μ} = \bar{X}, \hat{σ}² = (1/n)∑(X_i - \bar{X})².

Пример 2: Экспоненциальное распределение

X₁, ..., Xₙ ~ Exp(λ).

ММП-оценка: \hat{λ} = 1/\bar{X}.

Пример 3: Распределение Пуассона

X₁, ..., Xₙ ~ Poisson(λ).

ММП-оценка: \hat{λ} = \bar{X}.

Информационная матрица Фишера

Определения

Три эквивалентных определения:

  1. Через вторые производные логарифма правдоподобия
  2. Через вариацию score-функции
  3. Через ковариацию градиента логарифма правдоподобия

Свойства

  • I(θ) симметрична и положительно определена
  • I(θ) = E[S(θ)S(θ)ᵀ], где S(θ) - score-функция
  • Для независимых наблюдений: I_n(θ) = nI₁(θ)

Граница Крамера-Рао

Формулировка

Для любой несмещённой оценки \tilde{θ} параметра θ:

\[\text{Var}(\tilde{\theta}) \geq I^{-1}(\theta)\]

Асимптотическая версия

Для любой состоятельной оценки:

\[\liminf_{n \to \infty} n \cdot \text{Var}(\tilde{\theta}_n) \geq I^{-1}(\theta_0)\]

Практические аспекты

Численная оптимизация

  • Метод Ньютона-Рафсона
  • Метод градиентного подъёма
  • Алгоритмы EM для скрытых переменных

Проблемы

  • Множественные локальные максимумы
  • Граничные значения параметров
  • Численная неустойчивость

Диагностика

  • Проверка условий регулярности
  • Анализ информационной матрицы
  • Бутстреп-оценки дисперсии
