Теория чисел

Критерий Эйлера

Позволяет проверить квадратичность вычета по модулю простого числа.

Формулировка

Для нечетного простого p и числа a, не делящегося на p, значение степени (p-1)/2 определяет символ Лежандра.

\[a^{\frac{p-1}{2}}\equiv\left(\frac{a}{p}\right)\pmod p.\]

Идея доказательства

  1. Группа ненулевых классов по модулю p циклическая.
  2. Квадраты образуют подгруппу индекса два.
  3. Сравнение с символом Лежандра дает формулу критерия.

Следствия и примеры

  • Быстрый тест на квадратичный вычет.
  • База для закона квадратичной взаимности.
  • Связь с решением сравнения x^2 = a по модулю p.

Применения

  • Криптография на квадратичных вычетах.
  • Алгоритмы вычисления квадратных корней по модулю простого.
  • Тесты в вычислительной теории чисел.
