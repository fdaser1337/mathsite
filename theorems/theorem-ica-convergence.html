Обработка сигналов

Теорема о сходимости ICA

Сходимость анализа независимых компонент и разделение смешанных сигналов.

Основная модель ICA

Наблюдаем случайный вектор x ∈ ℝ^d, который является линейной смесью независимых компонент:

Модель смешивания

\[x = As\]

где:

  • s ∈ ℝ^d - вектор независимых компонент (источников)
  • A ∈ ℝ^{d×d} - матрица смешивания (неизвестная, невырожденная)
  • x ∈ ℝ^d - наблюдаемые смешанные сигналы

Задача ICA

Найти матрицу разделения W = A^{-1} такую, что:

\[y = Wx = WAs = s\]

Теорема об идентифицируемости ICA

Формулировка

Модель ICA идентифицируема (с точностью до масштаба и порядка) если:

  1. Компоненты s_i статистически независимы
  2. Максимум одна компонента имеет гауссово распределение
  3. Матрица A имеет полный ранг

Неопределённость

  • Масштаб: можно умножить s_i на константу и разделить соответствующий столбец A
  • Порядок: можно переставить компоненты s и соответствующие столбцы A

Интуиция

Гауссово распределение сферически симметрично, поэтому вращение не меняет независимость.

Центральная предельная теорема и ICA

Связь с ЦПТ

Сумма независимых случайных величин "более гауссова", чем исходные величины.

Негауссовость как мера независимости

Если y = w^T x = w^T As = ∑ w_i a_i s_i, то:

  • Если w параллелен одной строке A^{-1}, то y пропорциональна одной компоненте s_i
  • Если w является комбинацией строк, то y - смесь независимых компонент
  • По ЦПТ, смесь "более гауссова", чем отдельные компоненты

Максимизация негауссовости

Находим w, максимизирующий негауссовость y = w^T x.

Меры негауссовости

Эксцесс (kurtosis)

\[\text{kurt}(y) = E[y^4] - 3(E[y^2])^2\]
  • Супергауссов: kurt(y) > 0 (острый пик, тяжёлые хвосты)
  • Субгауссов: kurt(y) < 0 (плоская вершина, лёгкие хвосты)
  • Гауссов: kurt(y) = 0

Негэнтропия

\[J(y) = H(y_{gauss}) - H(y)\]

где H(y) - дифференциальная энтропия, y_{gauss} - гауссова случайная величина с той же дисперсией.

Аппроксимации негэнтропии

\[J(y) \approx \frac{1}{12}E[y^3]^2 + \frac{1}{48}(\text{kurt}(y))^2\]

Алгоритм FastICA

Фиксированная точка

Итерационная формула для одной компоненты:

\[w^{+} = E[xg(w^T x)] - E[g'(w^T x)]w\] \[w = \frac{w^{+}}{\|w^{+}\|}\]

где g - нелинейная функция (например, g(u) = tanh(u) или g(u) = u³).

Сходимость

Алгоритм сходится кубически вблизи оптимума.

Многокомпонентный случай

Последовательное нахождение компонент с ортогонализацией (например, грамм-шмидтом).

Теорема о сходимости FastICA

Формулировка

При определённых условиях алгоритм FastICA сходится к точке, где y = w^T x имеет максимальную негауссовость.

Условия сходимости

  • Функция g непрерывно дифференцируема
  • E[g'(w^T x)] > 0
  • Начальное приближение не ортогонально оптимальному решению

Свойства сходимости

  • Локальная сходимость гарантируется
  • Глобальная сходимость зависит от начальных условий
  • Кубическая сходимость вблизи оптимума

Метод максимального правдоподобия ICA

Логарифм правдоподобия

\[\log L(W) = \sum_{i=1}^{n} \sum_{j=1}^{d} \log p_j(w_j^T x_i) + n \log |\det W|\]

где p_j - плотности распределения источников.

Градиентный подъём

\[\frac{\partial \log L}{\partial W} = n(W^{-T})^T - \sum_{i=1}^{n} g(Wx_i)x_i^T\]

где g = (p₁'/p₁, ..., p_d'/p_d)^T.

Предобработка данных

Центрирование

\[x_{centered} = x - E[x]\]

Отбеливание (whitening)

\[x_{white} = V D^{-1/2} V^T x_{centered}\]

где V и D - собственные векторы и значения ковариационной матрицы.

Свойства отбелённых данных

  • E[x_{white} x_{white}^T] = I
  • Матрица разделения становится ортогональной
  • Ускорение сходимости

Выбор нелинейных функций

Супергауссовые источники

  • g(u) = tanh(u) - хорошая общая функция
  • g(u) = u³ - для симметричных распределений

Субгауссовые источники

  • g(u) = tanh(u) - работает хорошо
  • g(u) = u exp(-u²/2) - специально для субгауссовых

Устойчивые функции

g(u) = tanh(a u) с адаптивным параметром a.

Оценка качества разделения

Амплитудная шкала корреляции (SIR)

\[\text{SIR} = \frac{\|y_i\|_2}{\|y_i - s_{\pi(i)}\|_2}\]

Коэффициент разделения

\[\text{sep} = \sum_{i=1}^{d} \left(\sum_{j=1}^{d} \frac{|c_{ij}|}{\max_k |c_{ik}|} - 1\right)\]

где C = W A - матрица глобального разделения.

Связь с другими методами

PCA vs ICA

  • PCA: декоррелирует компоненты (вторые моменты)
  • ICA: делает компоненты независимыми (все моменты)
  • PCA - линейный метод, ICA - нелинейный оптимизации

Факторный анализ

ICA можно рассматривать как факторный анализ с негауссовыми факторами.

Разреженное кодирование

Оба метода ищут разреженные/независимые представления.

Практические аспекты

Число компонент

  • Не может превышать число наблюдений
  • Выбор основан на анализе собственных значений

Шум

  • ICA чувствителен к шуму
  • Предварительная фильтрация может улучшить результаты

Вычислительная сложность

  • FastICA: O(d²n) на итерацию
  • Метод максимального правдоподобия: O(d³n)

Расширения ICA

Ядерный ICA

Использование ядерных методов для нелинейного ICA.

Временной ICA

Учёт временной структуры сигналов.

Негативный ICA

Использование только неотрицательных компонент.

Применения

1. Обработка сигналов

Разделение аудио источников, "коктейльная вечеринка".

2. Обработка изображений

Удаление артефактов, сжатие.

3. Биомедицинские сигналы

Анализ ЭЭГ, ЭКГ, МЭГ.

4. Финансы

Выделение независимых факторов в временных рядах.
