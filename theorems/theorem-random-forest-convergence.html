Теорема о сходимости случайных лесов

Обобщающая способность ансамблей деревьев решений и декоррелирующий эффект.

Основная идея случайных лесов

Случайный лес - ансамбль деревьев решений, обученных с бэггингом и случайным выбором признаков.

Бэггинг (Bootstrap Aggregating)

Для каждого дерева t = 1, ..., B:

  1. Создаём бутстрап-выборку {(x_i^t, y_i^t)}_{i=1}^n из исходных данных
  2. Обучаем дерево h_t на этой выборке

Случайный выбор признаков

При каждом расщеплении выбираем случайное подмножество из m признаков.

Ансамбль

\[H(x) = \frac{1}{B}\sum_{t=1}^{B} h_t(x)\]

Теорема о декоррелированности

Формулировка

Пусть деревья h₁, ..., h_B имеют среднюю корреляцию ρ и дисперсию σ². Тогда дисперсия ансамбля:

\[\text{Var}\left(\frac{1}{B}\sum_{t=1}^{B} h_t\right) = \frac{1 - \rho}{B}\sigma^2 + \rho\sigma^2\]

Следствие

При B → ∞:

\[\text{Var}(H) \to \rho\sigma^2\]

Случайный выбор признаков уменьшает ρ, тем самым уменьшая дисперсию ансамбля.

Теорема о сходимости бэггинга

Условия

Пусть:

  • Данные (X, Y) независимы и одинаково распределены
  • Базовые алгоритмы h_t обучаются на независимых выборках
  • E[h_t(X)] = f*(X) для некоторой функции f*

Результат

\[E[(H(X) - f^*(X))^2] \leq \frac{1}{B}E[(h_1(X) - f^*(X))^2]\]

Бэггинг уменьшает дисперсию в B раз при независимых базовых алгоритмах.

Обобщающая способность

Разложение ошибки

\[E[(H(X) - Y)^2] = \text{Bias}^2 + \text{Variance} + \text{Noise}\]

Эффект случайного леса

  • Уменьшение дисперсии: за счёт усреднения деревьев
  • Уменьшение корреляции: за счёт случайного выбора признаков
  • Сохранение смещения: смещение отдельного дерева

Теорема о количестве признаков

Оптимальное m

Для p признаков рекомендуется:

  • Классификация: m = √p
  • Регрессия: m = p/3

Теоретическое обоснование

Баланс между:

  • Силой отдельных деревьев (больше m → сильнее деревья)
  • Декоррелированностью деревьев (меньше m → меньше корреляция)

Сходимость по числу деревьев

Теорема

При B → ∞ ошибка случайного леса сходится к пределу:

\[\lim_{B \to \infty} E[(H_B(X) - Y)^2] = \rho\sigma^2 + \text{Bias}^2 + \text{Noise}\]

Практические рекомендации

  • Ошибка стабилизируется при B ≈ 100-500
  • Дальнейшее увеличение даёт незначительное улучшение
  • Вычислительные затраты растут линейно с B

Out-of-Bag оценка

Идея

Каждая точка данных используется в обучении примерно 63% деревьев.

OOB ошибка

\[\text{OOB error} = \frac{1}{n}\sum_{i=1}^{n} \frac{1}{|B_i|}\sum_{t \in B_i} L(y_i, h_t(x_i))\]

где B_i - множество деревьев, не использующих x_i в обучении.

Свойства

OOB оценка является несмещённой оценкой ошибки обобщения.

Важность признаков

Перемешивание (Permutation Importance)

\[I_j = \frac{1}{B}\sum_{t=1}^{B} \left(L(y, h_t(X_{\text{perm}_j})) - L(y, h_t(X))\right)\]

Уменьшение примеси (Mean Decrease Impurity)

\[I_j = \frac{1}{B}\sum_{t=1}^{B}\sum_{\text{nodes } n \text{ with feature } j} p_n \Delta_n\]

где p_n - доля точек в узле, Δ_n - уменьшение нечистоты.

Теоретические границы

Граница обобщения

\[R(H) \leq R_{\text{emp}}(H) + \sqrt{\frac{c \log(B) + \log(1/\delta)}{n}}\]

Концентрация

С вероятностью 1-δ:

\[|R(H) - R_{\text{emp}}(H)| \leq O\left(\sqrt{\frac{\log(B) + \log(1/\delta)}{n}}\right)\]

Сравнение с другими методами

Случайный лес vs Бэггинг

  • Случайный лес добавляет случайный выбор признаков
  • Уменьшает корреляцию между деревьями
  • Лучше работает на высокоразмерных данных

Случайный лес vs Бустинг

  • Случайный лес: параллельное обучение, уменьшение дисперсии
  • Бустинг: последовательное обучение, уменьшение смещения
  • Случайный лес более устойчив к выбросам

Практические аспекты

Параметры

  • n_estimators: количество деревьев B
  • max_features: количество признаков m
  • max_depth: максимальная глубина деревьев
  • min_samples_split: минимальное число образцов для расщепления

Вычислительная сложность

  • Обучение: O(B × n × log n × m)
  • Предсказание: O(B × log n)
  • Параллелизация: деревья можно обучать независимо

Применения

1. Биоинформатика

Классификация генов, анализ белковых структур.

2. Финансы

Кредитный скоринг, оценка рисков.

3. Медицина

Диагностика заболеваний, анализ медицинских изображений.

4. Экология

Классификация видов, анализ экологических данных.
