Теорема о мажорируемой сходимости

Для мажорируемой последовательности измеримых функций можно менять предел и интеграл.

Формулировка

Пусть (X, Σ, μ) - пространство с мерой, {fₙ} - последовательность измеримых функций такая, что:

  1. fₙ(x) → f(x) поточечно при n → ∞
  2. Существует интегрируемая функция g такая, что |fₙ(x)| ≤ g(x) для всех n и почти всех x ∈ X

Тогда:

\[\lim_{n \to \infty} \int_X f_n d\mu = \int_X f d\mu\]

Интуитивное объяснение

Если последовательность функций сходится и все они «ограничены» одной интегрируемой функцией, то «площади под графиками» сходятся к площади под графиком предела.

Аналогия: Если у вас есть последовательность контейнеров, все помещающихся в один большой контейнер, и их формы сходятся к предельной форме, то их объёмы сходятся к объёму предельного контейнера.

Доказательство

Применим лемму Фату к последовательностям g + fₙ и g - fₙ:

\[\int_X \liminf (g + f_n) d\mu \leq \liminf \int_X (g + f_n) d\mu\] \[\int_X \liminf (g - f_n) d\mu \leq \liminf \int_X (g - f_n) d\mu\]

Поскольку g + fₙ → g + f и g - fₙ → g - f:

\[\int_X (g + f) d\mu \leq \int_X g d\mu + \liminf \int_X f_n d\mu\] \[\int_X (g - f) d\mu \leq \int_X g d\mu - \limsup \int_X f_n d\mu\]

Вычитая ∫g из обоих неравенств:

\[\int_X f d\mu \leq \liminf \int_X f_n d\mu\] \[-\int_X f d\mu \leq -\limsup \int_X f_n d\mu\]

Отсюда следует:

\[\limsup \int_X f_n d\mu \leq \int_X f d\mu \leq \liminf \int_X f_n d\mu\]

Поскольку всегда lim inf ≤ lim sup, получаем равенство.

Примеры

Пример 1: Сходящаяся последовательность

Пусть X = [0,1], μ - мера Лебега, fₙ(x) = xⁿ.

Тогда fₙ(x) → 0 для x ∈ [0,1), fₙ(1) = 1.

Мажоранта: g(x) = 1, так как |xⁿ| ≤ 1.

\[\int_0^1 x^n dx = \frac{1}{n+1} \to 0 = \int_0^1 0 dx\]

Пример 2: Тригонометрические функции

Пусть fₙ(x) = sin(nx)/n на [0,2π].

Тогда fₙ(x) → 0 и |fₙ(x)| ≤ 1/n ≤ 1.

\[\int_0^{2\pi} \frac{\sin(nx)}{n} dx = 0 \to 0\]

Пример 3: Без мажоранты

Пусть fₙ(x) = n·χ_{[0,1/n]}(x) на [0,1].

Тогда fₙ(x) → 0, но ∫fₙ = 1 для всех n.

Теорема неприменима, так как нет интегрируемой мажоранты.

Важность условий

1. Мажоранта существенна

Без мажоранты теорема неверна (см. пример 3).

2. Интегрируемость мажоранты

Функция g должна быть интегрируемой (∫|g| < ∞).

3. Поточечная сходимость

Требуется сходимость почти всюду.

Следствия

1. Замкнутость L¹

Если fₙ ∈ L¹, fₙ → f поточечно и ||fₙ||₁ ≤ M, то f ∈ L¹.

2. Непрерывность интеграла

Интеграл непрерывен относительно поточечной сходимости с мажорантой.

3. Теорема Лебега о дифференцировании

Используется в доказательстве теоремы о дифференцировании интеграла по параметру.

Обобщения

Сходимость по мере

Теорема верна для сходимости по мере (после выделения подпоследовательности).

Вектор-значный случай

Обобщение на функции со значениями в банаховых пространствах.

Вероятностная версия

Если Xₙ → X почти наверное и |Xₙ| ≤ Y с E[Y] < ∞, то E[Xₙ] → E[X].

Связь с другими теоремами

Лемма Фату

Используется в доказательстве теоремы о мажорируемой сходимости.

Теорема о монотонной сходимости

Частный случай, когда мажоранта - предельная функция.

Теорема Витали

Обобщение на семейства функций с равномерной интегрируемостью.

Применения

1. Теория вероятностей

Математические ожидания, сходимость случайных величин.

2. Функциональный анализ

L¹-пространства, теория операторов, слабая сходимость.

3. Гармонический анализ

Преобразования Фурье, свёртки, ряды Фурье.

4. Уравнения в частных производных

Вариационные методы, минимизация функционалов.

Историческая справка

Теорема доказана Анри Лебегом в начале XX века. Является одним из важнейших результатов теории интеграла Лебега.

Название «мажорируемая сходимость» отражает ключевое условие наличия мажоранты.
