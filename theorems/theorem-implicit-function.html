Теорема о неявной функции

Позволяет выразить одну переменную через другую из уравнения F(x,y) = 0 при определённых условиях.

Формулировка

Пусть функция F(x,y) непрерывно дифференцируема в окрестности точки (a,b) и F(a,b) = 0. Если частная производная по y не равна нулю:

\[\frac{\partial F}{\partial y}(a,b) \neq 0\]

то существует окрестность U точки a, в которой существует единственная непрерывно дифференцируемая функция φ(x) такая, что:

\[F(x, \phi(x)) = 0 \quad \text{для всех} \quad x \in U\]

Формула для производной

Производная неявной функции вычисляется по формуле:

\[\phi'(x) = -\frac{F_x(x, \phi(x))}{F_y(x, \phi(x))}\]

где F_x и F_y - частные производные F по x и y соответственно.

Геометрическая интерпретация

Уравнение F(x,y) = 0 определяет кривую в плоскости. Теорема гарантирует, что если в точке (a,b) касательная к этой кривой не вертикальна (F_y ≠ 0), то вблизи этой точки кривую можно представить как график функции y = φ(x).

Аналогия: Если вы смотрите на гору сбоку и в некоторой точке склон не вертикальный, то локально этот склон можно описать как функцию высоты от горизонтального расстояния.

Многомерный случай

Для системы уравнений F(x₁,...,xₙ,y₁,...,yₘ) = 0 с матрицей Якоби:

\[J = \begin{pmatrix} \frac{\partial F_1}{\partial y_1} & \cdots & \frac{\partial F_1}{\partial y_m} \\ \vdots & \ddots & \vdots \\ \frac{\partial F_m}{\partial y_1} & \cdots & \frac{\partial F_m}{\partial y_m} \end{pmatrix}\]

Если det(J) ≠ 0, то можно выразить y₁,...,yₘ как функции от x₁,...,xₙ.

Примеры

Пример 1: Окружность

Уравнение окружности: F(x,y) = x² + y² - 1 = 0.

Частные производные: F_x = 2x, F_y = 2y.

В точке (0,1): F_y = 2 ≠ 0, поэтому можно выразить y как функцию x:

\[y = \sqrt{1 - x^2}, \quad y' = -\frac{x}{y} = -\frac{x}{\sqrt{1 - x^2}}\]

Пример 2: Более сложное уравнение

F(x,y) = x³ + y³ - 3xy = 0 (декартов лист).

F_x = 3x² - 3y, F_y = 3y² - 3x.

В точке (1,1): F_y = 0, поэтому теорема неприменима (вертикальная касательная).

В точке (0,0): F_y = 0, теорема также неприменима.

Условия применимости

1. Непрерывная дифференцируемость

Функция F должна иметь непрерывные частные производные.

2. Ненулевая частная производная

F_y(a,b) ≠ 0 для выражения y через x (или F_x ≠ 0 для x через y).

3. Выполнение уравнения

F(a,b) = 0 - точка должна лежать на кривой.

Применения

1. Дифференциальная геометрия

Описание кривых и поверхностей как графиков функций.

2. Оптимизация

Метод множителей Лагранжа, анализ ограничений.

3. Экономика

Кривые безразличия, производственные функции.

4. Физика

Уравнения состояния, фазовые диаграммы.
