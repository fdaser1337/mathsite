Теорема о сходимости метода градиентного спуска

Условия сходимости и скорость сходимости градиентного спуска для выпуклых функций.

Формулировка

Пусть f: ℝⁿ → ℝ - выпуклая дифференцируемая функция с константой Липшица L для градиента:

\[\|\nabla f(x) - \nabla f(y)\| \leq L\|x - y\|\]

Предположим также, что f имеет минимум (существует x* такой, что ∇f(x*) = 0). Тогда для градиентного спуска с шагом 0 < α ≤ 1/L:

\[x_{k+1} = x_k - \alpha \nabla f(x_k)\]

последовательность {xₖ} сходится к точке минимума x*.

Скорость сходимости

Линейная сходимость для сильно выпуклых функций

Если f сильно выпукла с константой μ > 0:

\[f(y) \geq f(x) + \langle \nabla f(x), y-x\rangle + \frac{\mu}{2}\|y-x\|^2\]

то скорость сходимости линейная:

\[\|x_{k+1} - x^*\| \leq \left(1 - \alpha\mu\right)\|x_k - x^*\|\]

Сублинейная сходимость для выпуклых функций

Для просто выпуклых функций сходимость сублинейная:

\[f(x_k) - f(x^*) \leq \frac{\|x_0 - x^*\|^2}{2\alpha k}\]

Оптимальный выбор шага

Фиксированный шаг

Оптимальный фиксированный шаг: α = 1/L.

Адаптивный шаг

Метод наискорейшего спуска:

\[\alpha_k = \arg\min_{\alpha \geq 0} f(x_k - \alpha \nabla f(x_k))\]

Условие Вольфа

Шаг выбирается из условий:

\[f(x_k - \alpha_k \nabla f(x_k)) \leq f(x_k) - c_1\alpha_k\|\nabla f(x_k)\|^2\] \[|\langle \nabla f(x_k - \alpha_k \nabla f(x_k)), \nabla f(x_k)\rangle| \leq c_2\|\nabla f(x_k)\|^2\]

Доказательство сходимости

Свойство убывания функции

Используем разложение Тейлора:

\[f(x_{k+1}) \leq f(x_k) + \langle \nabla f(x_k), x_{k+1} - x_k\rangle + \frac{L}{2}\|x_{k+1} - x_k\|^2\]

Подставляя x_{k+1} = x_k - α∇f(x_k), получаем:

\[f(x_{k+1}) \leq f(x_k) - \alpha\|\nabla f(x_k)\|^2 + \frac{L\alpha^2}{2}\|\nabla f(x_k)\|^2\]

Оптимальный выбор α

Правая часть минимизируется при α = 1/L, что даёт:

\[f(x_{k+1}) \leq f(x_k) - \frac{1}{2L}\|\nabla f(x_k)\|^2\]

Примеры

Пример 1: Квадратичная функция

f(x) = ½xᵀAx + bᵀx + c, где A - положительно определённая матрица.

Градиентный спуск с оптимальным шагом сходится линейно.

Пример 2: Логистическая регрессия

f(w) = -∑[yᵢlog(σ(wᵀxᵢ)) + (1-yᵢ)log(1-σ(wᵀxᵢ))]

Градиентный спуск сходится сублинейно.

Пример 3: Невыпуклая функция

Для f(x) = x⁴ - x² градиентный спуск может застрять в локальном минимуме.

Модификации метода

Метод тяжёлого шара

\[x_{k+1} = x_k - \alpha\nabla f(x_k) + \beta(x_k - x_{k-1})\]

Метод Нестерова

\[y_{k+1} = x_k - \alpha\nabla f(x_k)\] \[x_{k+1} = y_{k+1} + \frac{k-1}{k+2}(y_{k+1} - y_k)\]

Стохастический градиентный спуск

\[x_{k+1} = x_k - \alpha_k \nabla f_{i_k}(x_k)\]

Сравнение с другими методами

Сравнение скоростей

  • Градиентный спуск: O(1/k) для выпуклых функций
  • Метод Нестерова: O(1/k²) для выпуклых функций
  • Метод Ньютона: квадратичная сходимость
  • Квази-ньютоновские методы: сверхлинейная сходимость

Вычислительные затраты

  • Градиентный спуск: O(n) на итерацию
  • Метод Ньютона: O(n³) на итерацию
  • Сопряжённые градиенты: O(n²) на итерацию

Практические аспекты

Критерии остановки

  • ‖∇f(xₖ)‖ < ε
  • |f(xₖ) - f(x_{k-1})| < ε
  • ‖xₖ - x_{k-1}‖ < ε
  • Максимальное число итераций

Проблемы

  • Выбор шага
  • Условия остановки оврагов
  • Численная устойчивость
  • Локальные минимумы для невыпуклых функций

Условия Липшица

Проверка условия Липшица

Для дважды дифференцируемых функций:

\[L \geq \max_{x} \lambda_{\max}(\nabla^2 f(x))\]

где λ_{\max} - максимальное собственное значение матрицы Гессе.

Оценка константы Липшица

На практике константу L можно оценить:

  • Теоретически через вторые производные
  • Эмпирически в процессе оптимизации
  • Используя backtracking line search

Применения

1. Машинное обучение

Обучение нейронных сетей, линейная регрессия, SVM.

2. Обработка сигналов

Сжатие, фильтрация, восстановление сигналов.

3. Оптимальное управление

Решение задач оптимального управления.

4. Финансы

Оптимизация портфелей, оценка рисков.

Историческая справка

Современная теория сходимости развита в XX веке.

Метод Нестерова (1983) значительно улучшил скорость сходимости.
