Линейная алгебра

Теорема о ранге и нулевом пространстве

Фундаментальная связь между рангом матрицы и размерностью её нулевого пространства.

Формулировка

Для матрицы A размера m×n:

\[\text{rank}(A) + \text{nullity}(A) = n\]

где rank(A) - ранг матрицы (размерность пространства образов), nullity(A) - размерность нулевого пространства (ядра).

Определения

Ранг матрицы

Максимальное число линейно независимых строк (или столбцов) матрицы.

Нулевое пространство (ядро)

Множество всех векторов x таких, что Ax = 0:

\[\text{ker}(A) = \{x \in \mathbb{R}^n : Ax = 0\}\]

Пространство образов

Множество всех векторов вида Ax:

\[\text{im}(A) = \{Ax : x \in \mathbb{R}^n\}\]

Геометрическая интерпретация

Линейное преобразование A: ℝⁿ → ℝᵐ «сжимает» пространство в двух направлениях:

  • rank(A) измеряет «распространение» в пространстве образов
  • nullity(A) измеряет «сжатие» в нулевом пространстве

Сумма этих размерностей равна исходной размерности n.

Пример вычисления

Матрица A = [[1, 2, 3], [2, 4, 6]]:

\[A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \end{pmatrix}\]

Ранг: Вторая строка = 2×первая, поэтому rank(A) = 1.

Нулевое пространство: Решаем Ax = 0:

\[x_1 + 2x_2 + 3x_3 = 0\] \[x_1 = -2x_2 - 3x_3\]

Базис нулевого пространства: {(-2, 1, 0), (-3, 0, 1)}, поэтому nullity(A) = 2.

Проверка: rank(A) + nullity(A) = 1 + 2 = 3 = n ✓

Следствия

1. Критерий обратимости

Квадратная матрица A обратима ⇔ rank(A) = n ⇔ nullity(A) = 0.

2. Существование решений

Система Ax = b имеет решение ⇔ rank(A) = rank(A|b).

3. Единственность решения

Если решение существует, оно единственно ⇔ nullity(A) = 0.

Применения

1. Решение систем линейных уравнений

Анализ существования и единственности решений.

2. Анализ данных

Метод главных компонент, анализ линейных зависимостей.

3. Компьютерная графика

Анализ линейных преобразований, проекции.

4. Машинное обучение

Анализ весовых матриц нейронных сетей.
