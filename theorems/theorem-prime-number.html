Теорема о простых числах

Асимптотический закон распределения простых чисел: π(x) ~ x/ln x.

Формулировка

Пусть π(x) - количество простых чисел, не превышающих x. Тогда:

\[\lim_{x \to \infty} \frac{\pi(x)}{x/\ln x} = 1\]

Это означает, что π(x) асимптотически равно x/ln x.

Более точная формулировка:

\[\pi(x) = \frac{x}{\ln x} + O\left(\frac{x}{\ln^2 x}\right)\]

Исторический контекст

Теорема была независимо сформулирована Гауссом (1792) и Лежандром (1798) на основе наблюдений за таблицами простых чисел.

Гаусс заметил, что плотность простых чисел около x примерно равна 1/ln x.

Доказана независимо Жаком Адамаром и Шарлем де ля Валле Пуссеном в 1896 году.

Интегральный логарифм

Более точная аппроксимация использует интегральный логарифм:

\[\text{li}(x) = \int_2^x \frac{dt}{\ln t}\]

Тогда:

\[\pi(x) = \text{li}(x) + O\left(x e^{-c\sqrt{\ln x}}\right)\]

где c > 0 - константа. Это гораздо более точная аппроксимация.

Связь с дзета-функцией Римана

Теорема о простых числах эквивалентна утверждению, что дзета-функция Римана ζ(s) не имеет нулей на прямой Re(s) = 1.

\[\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} = \prod_{p \text{ простое}} \frac{1}{1-p^{-s}}\]

Это показывает глубокую связь между распределением простых чисел и аналитическими свойствами ζ(s).

Числовые данные

Примеры значений π(x):

  • π(10) = 4, 10/ln 10 ≈ 4.34
  • π(100) = 25, 100/ln 100 ≈ 21.71
  • π(1000) = 168, 1000/ln 1000 ≈ 144.76
  • π(10⁶) = 78498, 10⁶/ln 10⁶ ≈ 72382.41
  • π(10⁹) = 50847534, 10⁹/ln 10⁹ ≈ 50847334.53

Относительная погрешность уменьшается с ростом x.

Следствия и приложения

1. n-е простое число

Если pₙ - n-е простое число, то:

\[p_n \sim n \ln n\]

2. Промежутки между простыми

Средний промежуток между простыми числами около x равен ln x.

3. Вероятностная интерпретация

Вероятность того, что случайное число n ≤ x простое, примерно равна 1/ln x.

4. Алгоритмическая сложность

Оценка сложности алгоритмов факторизации и тестирования простоты.

Методы доказательства

Аналитический метод

Основан на комплексном анализе дзета-функции Римана.

Ключевые шаги:

  1. Доказательство отсутствия нулей ζ(s) на Re(s) = 1
  2. Использование формул для π(x) через интегралы от ζ(s)
  3. Оценка остаточных членов

Элементарные методы

Сельберг (1948) и Эрдёш (1949) дали элементарные доказательства без использования комплексного анализа.

Обобщения и гипотезы

Гипотеза Римана

Все нетривиальные нули ζ(s) лежат на прямой Re(s) = 1/2.

Это дало бы гораздо более точные оценки для π(x).

Связь с другими теоремами

Теорема Дирихле о простых в прогрессиях

В любой арифметической прогрессии an + b с (a,b) = 1 бесконечно много простых.

Теорема Чебышёва

Существуют константы c₁, c₂ такие, что c₁x/ln x ≤ π(x) ≤ c₂x/ln x.

Теорема Грина-Тао

Существуют произвольно длинные арифметические прогрессии простых чисел.

Тесты простоты, алгоритмы факторизации.

3. Статистика

Моделирование распределения простых чисел.
