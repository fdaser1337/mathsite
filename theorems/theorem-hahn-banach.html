Функциональный анализ

Теорема Хана-Банаха

Любой линейный функционал можно продолжить с сохранением нормы на всё пространство.

Формулировка

Пусть X - нормированное пространство, Y ⊂ X - подпространство, f: Y → ℝ - непрерывный линейный функционал. Тогда существует непрерывный линейный функционал F: X → ℝ такой, что:

  1. F(y) = f(y) для всех y ∈ Y (продолжение)
  2. ‖F‖ = ‖f‖ (сохранение нормы)

где ‖f‖ = sup{|f(y)| : y ∈ Y, ‖y‖ ≤ 1} - норма функционала.

Геометрическая интерпретация

Теорема позволяет «растянуть» линейный функционал с подпространства на всё пространство без увеличения его «наклона».

Аналогия: Если у вас есть линейная функция, определённая на прямой, вы можете продолжить её на всю плоскость так, чтобы она оставалась линейной и не стала «круче».

Важные следствия

1. Существование достаточного числа функционалов

Для любого ненулевого x ∈ X существует функционал f ∈ X* такой, что f(x) = ‖x‖ и ‖f‖ = 1.

2. Разделяющие семейства

Пространство функционалов X* разделяет точки X: если x₁ ≠ x₂, то существует f ∈ X* с f(x₁) ≠ f(x₂).

3. Теорема об отделимости

Любые непересекающиеся выпуклые множества в банаховом пространстве можно разделить гиперплоскостью.

Примеры

Пример 1: ℝ²

Пусть X = ℝ², Y = {(x,0) : x ∈ ℝ}, f(x,0) = 2x.

Продолжение: F(x,y) = 2x (норма сохраняется).

Пример 2: Пространство C[0,1]

Пусть X = C[0,1] с нормой ‖f‖ = max|f(t)|, Y = {константные функции}, g(c) = c.

Продолжение: F(f) = f(0) (оценка в точке).

Пример 3: L²[0,1]

Пусть Y - подпространство функций, равных нулю на [0,1/2], f(y) = ∫₀¹ y(t)dt.

Продолжение: F(x) = ∫₀¹ x(t)dt (сохранение нормы).

Доказательство (основная идея)

Используется лемма Цорна о максимальных элементах:

  1. Рассматриваем все возможные продолжения f на большие подпространства
  2. Упорядочиваем их по включению областей определения
  3. По лемме Цорна существует максимальное продолжение
  4. Показываем, что максимальное продолжение определено на всём X

Ключевая техническая лемма: если функционал можно продолжить на одномерное расширение, то можно продолжить с сохранением нормы.

Применения

1. Функциональный анализ

Двойственность, теория операторов, спектральная теория.

2. Теория оптимизации

Условия оптимальности, теорема Куна-Таккера.

3. Экономика

Теория цен, равновесие, благосостояние.

4. Квантовая механика

Теорема о представлении, измеримые наблюдаемые.

Обобщения

Комплексный случай

Теорема верна и для комплексных пространств с соответствующими модификациями.

Локально-выпуклые пространства

Обобщение на более общие классы топологических векторных пространств.

Вектор-значные функционалы

Продолжение операторов в банаховы пространства.

Историческая справка

Теорема доказана независимо Хансом Ханом (1927) и Стефаном Банахом (1929). Является одной из фундаментальных теорем функционального анализа.

Имеет многочисленные применения в различных разделах математики и её приложениях.
