Дифференциальные уравнения

Теорема Гронуолла

Оценка функций, удовлетворяющих интегральным или дифференциальным неравенствам.

Формулировка (дифференциальная версия)

Пусть u: [a,b] → [0,∞) - непрерывная функция, удовлетворяющая неравенству:

\[u'(t) \leq \beta(t) u(t) + \alpha(t)\]

для всех t ∈ [a,b], где α, β - непрерывные функции. Тогда:

\[u(t) \leq u(a) e^{\int_a^t \beta(s) ds} + \int_a^t \alpha(s) e^{\int_s^t \beta(\tau) d\tau} ds\]

Формулировка (интегральная версия)

Пусть u: [a,b] → [0,∞) - непрерывная функция, удовлетворяющая неравенству:

\[u(t) \leq c + \int_a^t [\alpha(s) u(s) + \beta(s)] ds\]

где c ≥ 0, α, β - непрерывные функции, α(t) ≥ 0. Тогда:

\[u(t) \leq c e^{\int_a^t \alpha(s) ds} + \int_a^t \beta(s) e^{\int_s^t \alpha(\tau) d\tau} ds\]

Частный случай (классическая теорема Гронуолла)

Если u(t) ≥ 0 и u(t) ≤ c + k∫ₐᵗ u(s) ds, где c, k ≥ 0 - константы, то:

\[u(t) \leq c e^{k(t-a)}\]

Доказательство (интегральной версии)

Определим функцию:

\[v(t) = \int_a^t \alpha(s) u(s) ds\]

Тогда v'(t) = α(t)u(t) и из неравенства следует:

\[u(t) \leq c + v(t) + \int_a^t \beta(s) ds\]

Следовательно:

\[v'(t) = \alpha(t) u(t) \leq \alpha(t) [c + v(t) + \int_a^t \beta(s) ds]\]

Решая это линейное дифференциальное неравенство, получаем искомую оценку.

Примеры

Пример 1: Оценка решения ОДУ

Пусть y' = y + 1, y(0) = 0. Решение: y(t) = eᵗ - 1.

Из теоремы Гронуолла: |y(t)| ≤ |y(0)|eᵗ + ∫₀ᵗ e^{t-s} ds = eᵗ - 1.

Пример 2: Устойчивость

Если |y'(t)| ≤ 2|y(t)| и y(0) = 1, то |y(t)| ≤ e^{2t}.

Пример 3: Системы уравнений

Для системы y' = Ay(t) + f(t) с ||A|| ≤ k:

\[\|y(t)\| \leq \|y(0)\| e^{kt} + \int_0^t \|f(s)\| e^{k(t-s)} ds\]

Применения

1. Единственность решений ОДУ

Используется в доказательстве теоремы Пикара-Линделёфа.

2. Устойчивость решений

Оценка роста решений и их чувствительности к начальным данным.

3. Теория управления

Оценка переходных процессов в системах управления.

4. Численные методы

Оценка погрешности численных методов решения ОДУ.

Обобщения

Неравенство Гронуолла-Беллмана

Для вектор-функций: ||u(t)|| ≤ c + ∫ₐᵗ α(s)||u(s)|| ds.

Неравенство Бихари

Обобщение на нелинейные функции: u(t) ≤ c + ∫ₐᵗ f(s,u(s)) ds.

Дискретная версия

1. Инженерные расчёты

Оценка максимальных значений в динамических системах.

2. Физика

Оценка энергии в колебательных системах.

3. Экономика

Оценка роста экономических моделей.

4. Биология

Оценка популяционной динамики.

Историческая справка

Теорема названа в честь Томаса Гронуолла (1877-1932), шведского математика. Первоначально доказана в 1919 году.

Ричард Беллман обобщил теорему в 1950-х годах, что привело к неравенству Гронуолла-Беллмана.

Является одним из фундаментальных инструментов в теории дифференциальных уравнений и их приложениях.
