Топология

Теорема о фундаментальной группе

Фундаментальная группа классифицирует петли в топологическом пространстве по гомотопии.

Формулировка

Пусть X - топологическое пространство, x₀ ∈ X - отмеченная точка. Фундаментальная группа π₁(X, x₀) - это множество классов гомотопии петель с началом и концом в x₀ с операцией композиции петель.

π₁(X, x₀) является группой, где:

  • Элементы - классы эквивалентности петель
  • Операция - композиция петель
  • Нейтральный элемент - константная петля
  • Обратный элемент - петля, проходимая в обратном направлении

Ключевые понятия

Петля

Непрерывное отображение γ: [0,1] → X такое, что γ(0) = γ(1) = x₀.

Гомотопия петель

Две петли γ₁, γ₂ гомотопны, если существует непрерывное отображение H: [0,1] × [0,1] → X такое, что:

\[H(s,0) = \gamma_1(s), \quad H(s,1) = \gamma_2(s)\] \[H(0,t) = H(1,t) = x_0 \text{ для всех } t\]

Композиция петель

Для петель γ₁, γ₂: (γ₁ * γ₂)(s) = γ₁(2s) при s ∈ [0,0.5], γ₂(2s-1) при s ∈ [0.5,1].

Примеры фундаментальных групп

1. Окружность S¹

π₁(S¹) ≅ ℤ - целые числа. Петли классифицируются числом оборотов.

2. Диск D²

π₁(D²) ≅ {0} - тривиальная группа. Любая петля стягивается в точку.

3. Тор T²

π₁(T²) ≅ ℤ × ℤ. Петли классифицируются двумя числами оборотов.

4. Проективная плоскость ℝP²

π₁(ℝP²) ≅ ℤ₂ - группа из двух элементов.

Свойства фундаментальной группы

1. Топологический инвариант

Гомеоморфные пространства имеют изоморфные фундаментальные группы.

2. Независимость от точки

Для линейно связного пространства π₁(X, x₀) ≅ π₁(X, x₁).

3. Функториальность

Непрерывное отображение f: X → Y индуцирует гомоморфизм f*: π₁(X) → π₁(Y).

Теорема о накрытии

Если p: Ŷ → X - универсальное накрытие, то π₁(X) действует на Ŷ как группа накрытия.

Пространство Ŷ односвязно (π₁(Ŷ) = {0}) и является «пространством путей» X.

Применения

1. Классификация поверхностей

Замкнутые поверхности классифицируются фундаментальными группами.

2. Теория узлов

Группа узла - фундаментальная группа дополнения узла.

3. Алгебраическая топология

Вычисление гомотопических групп, когомологий.

4. Физика

Теория струн, квантовая теория поля, топологические инварианты.

Вычисление π₁(S¹)

Используем накрывающее отображение exp: ℝ → S¹, exp(t) = e^(2πit).

Петля в S¹ поднимается в путь в ℝ. Конечная точка поднимается в начальную точку + n, где n - число оборотов.

Это устанавливает изоморфизм π₁(S¹) ≅ ℤ.

Связь с другими теориями

Гомологии

π₁(X) абелианизация ≅ H₁(X, ℤ) - первая группа гомологий.

Высшие гомотопические группы

πₙ(X) для n > 2 обобщают концепцию на сферы.

Теория расслоений

Фундаментальная группа действует на слоях расслоений.
