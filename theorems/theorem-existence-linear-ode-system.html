Дифференциальные уравнения

Теорема о существовании линейных ОДУ

Существование и структура фундаментальной системы решений для систем линейных обыкновенных дифференциальных уравнений.

Формулировка

Рассмотрим систему линейных однородных ОДУ:

\[\mathbf{y}' = A(x)\mathbf{y}\]

где A(x) - непрерывная n×n матрица на интервале I ⊂ ℝ.

Тогда:

  1. Существует n линейно независимых решений y₁(x), ..., yₙ(x)
  2. Любое решение является линейной комбинацией этих решений
  3. Существует фундаментальная матрица решений Φ(x) такая, что Φ'(x) = A(x)Φ(x)

Фундаментальная матрица решений

Фундаментальная матрица Φ(x) - это матрица, столбцы которой образуют фундаментальную систему решений.

Свойства фундаментальной матрицы:

  • Φ(x) невырождена для всех x ∈ I
  • Φ'(x) = A(x)Φ(x)
  • Если Φ₀ - другая фундаментальная матрица, то Φ₀(x) = Φ(x)C, где C - невырожденная постоянная матрица

Определитель Вронского

Определитель Вронского W(x) = det Φ(x) удовлетворяет уравнению Лиувилля:

\[W'(x) = \text{tr}(A(x)) \cdot W(x)\]

Решение:

\[W(x) = W(x_0) \exp\left(\int_{x_0}^x \text{tr}(A(t)) dt\right)\]

Если W(x₀) ≠ 0, то W(x) ≠ 0 для всех x ∈ I.

Метод построения решений

1. Матричная экспонента (постоянный коэффициент)

Если A(x) ≡ A (постоянная матрица), то:

\[\Phi(x) = e^{Ax} = \sum_{k=0}^{\infty} \frac{A^k x^k}{k!}\]

2. Метод вариации постоянных

Для неоднородной системы y' = A(x)y + f(x):

\[\mathbf{y}(x) = \Phi(x)\left(\int \Phi^{-1}(x)\mathbf{f}(x) dx + \mathbf{c}\right)\]

3. Степенной метод (аналитические коэффициенты)

Если A(x) аналитична, решения ищутся в виде степенных рядов.

Примеры

Пример 1: Постоянные коэффициенты

y' = Ay, где A = [[2,1],[1,2]].

Собственные значения: λ₁ = 3, λ₂ = 1.

Фундаментальная матрица:

\[\Phi(x) = \begin{bmatrix} e^{3x} & e^{x} \\ e^{3x} & -e^{x} \end{bmatrix}\]

Пример 2: Переменные коэффициенты

y' = [[0,1],[-1/x,0]]y.

Решения: y₁(x) = [cos(ln x), -sin(ln x)]ᵀ, y₂(x) = [sin(ln x), cos(ln x)]ᵀ.

Пример 3: Система второго порядка

y'' + y = 0 эквивалентно системе:

\[\begin{cases} y_1' = y_2 \\ y_2' = -y_1 \end{cases}\]

Фундаментальная матрица: Φ(x) = [[cos x, sin x],[-sin x, cos x]].

Неоднородная система

Для системы y' = A(x)y + f(x) общее решение:

\[\mathbf{y}(x) = \Phi(x)\mathbf{c} + \Phi(x)\int_{x_0}^x \Phi^{-1}(t)\mathbf{f}(t) dt\]

где Φ(x) - фундаментальная матрица соответствующей однородной системы.

Теорема существования и единственности

Для задачи Коши y'(x₀) = y₀:

\[\mathbf{y}(x) = \Phi(x)\Phi^{-1}(x_0)\mathbf{y}_0\]

Решение существует и единственно на всём интервале непрерывности A(x).

Применения

1. Физика

Колебательные системы, квантовая механика, электродинамика.

2. Теория управления

Системы управления, устойчивость, наблюдаемость.

3. Экономика

Модели экономической динамики, финансовые рынки.

4. Биология

Популяционная динамика, биохимические кинетики.

Численные методы

1. Метод Рунге-Кутты

Прямое численное интегрирование системы.

2. Метод матричной экспоненты

Вычисление e^{Ax} через разложения в ряд или диагонализацию.

3. Метод shooting

Для краевых задач преобразуются в задачи Коши.
