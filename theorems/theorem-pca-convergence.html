Теорема о сходимости PCA

Сходимость метода главных компонент и оптимальные свойства проекции.

Основная постановка PCA

Пусть X ∈ ℝ^{n×d} - матрица данных, где n - число наблюдений, d - число признаков.

Цель PCA

Найти ортогональную проекцию на k-мерное подпространство, минимизирующую ошибку реконструкции:

\[\min_{U \in \mathbb{R}^{d \times k}, U^T U = I} \frac{1}{n}\|X - XUU^T\|_F^2\]

Эквивалентная формулировка

Максимизация сохраненной дисперсии:

\[\max_{U \in \mathbb{R}^{d \times k}, U^T U = I} \frac{1}{n}\|XUU^T\|_F^2\]

Теорема об оптимальности PCA

Формулировка

Решением задачи PCA являются собственные векторы ковариационной матрицы Σ = (1/n)X^T X, соответствующие k наибольшим собственным значениям.

Доказательство

Используем разложение по собственным векторам:

\[X = U\Lambda V^T\]

где U ∈ ℝ^{n×n}, V ∈ ℝ^{d×d} - ортогональные матрицы, Λ - диагональная матрица сингулярных чисел.

Оптимальная проекция выбирается по первым k столбцам V.

Теорема о сходимости эмпирического PCA

Формулировка

Пусть X₁, ..., Xₙ - независимые выборки из распределения с ковариационной матрицей Σ. Тогда эмпирические главные компоненты сходятся к истинным.

Скорость сходимости

\[\|\hat{v}_j - v_j\| = O_p\left(\frac{1}{\sqrt{n}}\right)\]

где \hat{v}_j - эмпирический собственный вектор, v_j - истинный собственный вектор.

Условия

  • Собственные значения λ₁ > λ₂ > ... > λ_k (разделённость)
  • Конечные четвёртые моменты
  • Подчинённость распределения (subgaussian)

Теорема о концентрации собственных значений

Неравенство Дэвиса-Кахана

Для случайной матрицы X с независимыми строками:

\[P(\|\hat{\Sigma} - \Sigma\| \geq t) \leq 2d \exp\left(-cn t^2\right)\]

где \hat{Σ} = (1/n)X^T X - эмпирическая ковариация, c - константа.

Следствие для собственных значений

\[|\hat{\lambda}_j - \lambda_j| \leq \|\hat{\Sigma} - \Sigma\|\]

Итерационные методы PCA

Метод степеней

Для нахождения главного собственного вектора:

\[v^{(t+1)} = \frac{\Sigma v^{(t)}}{\|\Sigma v^{(t)}\|}\]

Сходимость метода степеней

\[\|v^{(t)} - v_1\| \leq C\left(\frac{\lambda_2}{\lambda_1}\right)^t\]

где λ₁ > λ₂ - два наибольших собственных значения.

Дефляционный метод

Последовательное нахождение собственных векторов с вычитанием проекций.

Случайный PCA

Идея

Использование случайных проекций для ускорения вычислений.

Алгоритм

  1. Генерируем случайную матрицу Ω ∈ ℝ^{d×k}
  2. Вычисляем Y = XΩ
  3. QR-разложение Y = QR
  4. Проектируем B = Q^T X
  5. SVD разложение B = U_B Σ_B V_B^T
  6. Искомые компоненты: U = QV_B

Теорема о точности

\[\|X - UU^T X\|_F \leq (1 + \varepsilon)\|X - X_k\|_F\]

с вероятностью ≥ 1-δ при k ≥ O(log(1/δ)/ε²).

Ядровой PCA

Отображение в пространство признаков

\[\phi: \mathbb{R}^d \to \mathcal{H}\]

Ядровая матрица

\[K_{ij} = \langle \phi(x_i), \phi(x_j) \rangle = k(x_i, x_j)\]

Собственные векторы в пространстве признаков

\[v_j = \sum_{i=1}^{n} \alpha_{ij} \phi(x_i)\]

где α_{ij} - собственные векторы центрированной матрицы K.

Разложение ошибки

Теорема о разложении

\[\frac{1}{n}\|X - X_k\|_F^2 = \sum_{j=k+1}^{d} \lambda_j\]

где X_k = XU_kU_k^T - проекция на первые k главных компонент.

Объяснённая дисперсия

\[\text{Объяснённая дисперсия} = \frac{\sum_{j=1}^{k} \lambda_j}{\sum_{j=1}^{d} \lambda_j}\]

Выбор числа компонент

Правило Кайзера

Выбирать компоненты с λ_j > 1 (для стандартизированных данных).

Метод локтя

Выбор точки изгиба на графике собственных значений.

Критерий накопленной дисперсии

Выбор минимального k такого, что:

\[\frac{\sum_{j=1}^{k} \lambda_j}{\sum_{j=1}^{d} \lambda_j} \geq \tau\]

где τ ∈ [0.8, 0.95] - порог объяснённой дисперсии.

Свойства PCA

Оптимальность

  • Минимизация ошибки реконструкции
  • Максимизация сохранённой дисперсии
  • Декоррелированность компонент

Ограничения

  • Линейность преобразования
  • Чувствительность к масштабированию
  • Интерпретируемость компонент

Связь с другими методами

Факторный анализ

PCA - частный случай факторного анализа с единичными уникальностями.

SVD

PCA через SVD центрированной матрицы данных.

ICA

ICA ищет независимые компоненты, PCA - некоррелированные.

Практические аспекты

Предобработка

  • Центрирование данных
  • Стандартизация признаков
  • Обработка выбросов

Вычислительная сложность

  • Полный PCA: O(min(nd², n²d))
  • Случайный PCA: O(ndk)
  • Итерационные методы: O(ndk) на итерацию

Применения

1. Снижение размерности

Визуализация данных, ускорение алгоритмов.

2. Сжатие данных

Хранение изображений, аудио.

3. Обработка сигналов

Удаление шума, выделение признаков.

4. Биоинформатика

Анализ экспрессии генов, ДНК-микрочипы.
