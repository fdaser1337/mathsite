Теорема о сходимости эмпирического распределения

Равномерная сходимость эмпирической функции распределения к истинной.

Основные определения

Пусть X₁, X₂, ..., Xₙ - независимые одинаково распределённые случайные величины с функцией распределения F(x) = P(X ≤ x).

Эмпирическая функция распределения

\[F_n(x) = \frac{1}{n}\sum_{i=1}^{n} \mathbf{1}_{\{X_i \leq x\}}\]

где 1_{A} - индикатор события A.

Статистика Колмогорова-Смирнова

\[D_n = \sup_{x \in \mathbb{R}} |F_n(x) - F(x)|\]

Теорема Гливенко-Кантелли

Формулировка

Для эмпирической функции распределения F_n(x) и истинной функции распределения F(x):

\[\sup_{x \in \mathbb{R}} |F_n(x) - F(x)| \xrightarrow{a.s.} 0\]

Это означает, что F_n(x) сходится к F(x) равномерно почти наверное.

Сильная сходимость

Теорема утверждает сходимость почти наверное (сильную сходимость), а не только по вероятности.

Доказательство теоремы

Основная идея

Используем закон больших чисел и неравенство Дворецки-Кифера-Вольфовица.

Шаги доказательства

  1. Для любого фиксированного x: F_n(x) → F(x) почти наверное по ЗБЧ
  2. Используем неравенство Дворецкого-Кифера-Вольфовица:
    3. \[P\left(\sup_{x \in \mathbb{R}} |F_n(x) - F(x)| > \varepsilon\right) \leq 2e^{-2n\varepsilon^2}\]
  3. Применяем лемму Бореля-Кантелли для получения сходимости почти наверное

Теорема Колмогорова

Формулировка

Если F(x) - непрерывная функция распределения, то:

\[\sqrt{n}D_n \xrightarrow{d} K\]

где K - распределение Колмогорова с функцией распределения:

\[P(K \leq x) = 1 - 2\sum_{k=1}^{\infty} (-1)^{k-1}e^{-2k^2x^2}, \quad x > 0\]

Следствие

Позволяет строить критерии согласия и доверительные полосы.

Неравенство Дворецки-Кифера-Вольфовица

Формулировка

Для любого ε > 0 и n ≥ 1:

\[P\left(\sup_{x \in \mathbb{R}} |F_n(x) - F(x)| > \varepsilon\right) \leq 2e^{-2n\varepsilon^2}\]

Значение

Даёт экспоненциальную оценку вероятности большого отклонения.

Является неравенством типа Хoeffding для эмпирического процесса.

Многомерный случай

Эмпирическое распределение в ℝ^d

\[F_n(x_1, ..., x_d) = \frac{1}{n}\sum_{i=1}^{n} \mathbf{1}_{\{X_{i1} \leq x_1, ..., X_{id} \leq x_d\}}\]

Теорема сходимости

В многомерном случае также имеет место равномерная сходимость почти наверное.

Сложности

  • Распределение статистики D_n становится сложным
  • Требуются специальные методы для вычисления критических значений

Применения

1. Критерии согласия

Критерий Колмогорова-Смирнова:

Проверка гипотезы H₀: F = F₀ против H₁: F ≠ F₀.

Статистика: D_n = sup|F_n(x) - F₀(x)|.

Отвергаем H₀, если D_n > c_α, где c_α - критическое значение.

2. Двувыборочный критерий

Проверка равенства двух распределений.

\[D_{n,m} = \sup_{x} |F_n(x) - G_m(x)|\]

3. Доверительные полосы

Построение доверительных полос для функции распределения:

\[F_n(x) \pm c_{\alpha}\frac{1}{\sqrt{n}}\]

Примеры

Пример 1: Равномерное распределение

X₁, ..., Xₙ ~ Uniform[0,1].

F(x) = x для x ∈ [0,1].

Эмпирическая функция распределения ступенчатая.

Пример 2: Нормальное распределение

X₁, ..., Xₙ ~ N(μ, σ²).

F(x) = Φ((x-μ)/σ).

Используется для проверки нормальности данных.

Пример 3: Экспоненциальное распределение

X₁, ..., Xₙ ~ Exp(λ).

F(x) = 1 - e^{-λx}.

Свойства эмпирического распределения

Несмещённость

\[E[F_n(x)] = F(x)\]

Дисперсия

\[\text{Var}(F_n(x)) = \frac{F(x)(1-F(x))}{n}\]

Ковариация

\[\text{Cov}(F_n(x), F_n(y)) = \frac{\min(F(x), F(y)) - F(x)F(y)}{n}\]

Связь с другими теоремами

Закон больших чисел

Для каждой точки x: F_n(x) → F(x) по вероятности.

Центральная предельная теорема

Для фиксированного x: √n(F_n(x) - F(x)) → N(0, F(x)(1-F(x))).

Теорема Донскера

Функциональная предельная теорема для эмпирического процесса.

Обобщения

Взвешенное эмпирическое распределение

\[F_n^w(x) = \frac{\sum_{i=1}^{n} w_i \mathbf{1}_{\{X_i \leq x\}}}{\sum_{i=1}^{n} w_i}\]

Кен-оценка функции распределения

Сглаженная версия эмпирической функции распределения.

Эмпирические меры в банаховых пространствах

Обобщение на функциональные данные.

Практические аспекты

Вычисление статистики D_n

  • Сортировка выборки: X_(1) ≤ X_(2) ≤ ... ≤ X_(n)
  • Вычисление максимумов в точках X_(i)
  • D_n = max(max|F_n(X_(i)) - F(X_(i))|, max|F_n(X_(i)-) - F(X_(i))|)

Критические значения

  • Табличные значения для стандартного случая
  • Бутстреп для сложных распределений
  • Метод Монте-Карло для многомерного случая
