Теорема Лиувилля

Всякая ограниченная аналитическая функция на всей комплексной плоскости является постоянной.

Формулировка

Если функция f аналитична на всей комплексной плоскости ℂ (целая функция) и существует константа M такая, что |f(z)| ≤ M для всех z ∈ ℂ, то f(z) ≡ const.

Доказательство

Используем формулу Коши для производных:

\[f'(z) = \frac{1}{2\pi i} \oint_{|\zeta-z|=R} \frac{f(\zeta)}{(\zeta-z)^2} d\zeta\]

Оценим модуль производной:

\[|f'(z)| \leq \frac{1}{2\pi} \oint_{|\zeta-z|=R} \frac{|f(\zeta)|}{|\zeta-z|^2} |d\zeta| \leq \frac{1}{2\pi} \cdot \frac{M}{R^2} \cdot 2\pi R = \frac{M}{R}\]

Так как это верно для любого R > 0, устремим R → ∞:

\[|f'(z)| \leq \lim_{R \to \infty} \frac{M}{R} = 0\]

Следовательно, f'(z) = 0 для всех z, поэтому f(z) ≡ const.

Обобщения

Обобщённая теорема Лиувилля

Если |f(z)| ≤ C|z|ⁿ для достаточно больших |z|, то f - многочлен степени не выше n.

Теорема Лиувилля для гармонических функций

Ограниченная гармоническая функция на ℝⁿ постоянна.

Примеры

Пример 1: Константы

f(z) = 5 - ограниченная аналитическая функция, постоянна по теореме.

Пример 2: Экспонента не ограничена

f(z) = eᶻ аналитична на ℂ, но |eᶻ| → ∞ при Re(z) → ∞, поэтому теорема неприменима.

Пример 3: Синус не ограничен

f(z) = sin(z) аналитична на ℂ, но |sin(iy)| = |sh(y)| → ∞ при y → ∞.

Пример 4: Функция с бесконечным числом нулей

Если f(z) = 0 для всех целых z, то f ≡ 0 (частный случай теоремы о единственности).

Следствия

1. Фундаментальная теорема алгебры

Из теоремы Лиувилля следует, что любой многочлен степени n ≥ 1 имеет корень.

2. Теорема о единственности

Если целая функция имеет бесконечно много нулев с предельной точкой, она тождественно равна нулю.

3. Принцип максимума

Модуль не константной аналитической функции не может достигать максимума внутри области.

Применения

1. Теория функций

Классификация целых функций, изучение их роста.

2. Дифференциальные уравнения

Анализ решений линейных дифференциальных уравнений.

3. Комплексная динамика

Изучение итераций аналитических функций.

4. Квантовая механика

Анализ волновых функций, рассеяние.

Теорема Римана об отображении

Любая односвязная область, отличная от всей плоскости, конформно эквивалентна единичному кругу.

Историческая справка

Теорема доказана Жозефом Лиувиллем в 1844 году. Является одним из фундаментальных результатов комплексного анализа.
