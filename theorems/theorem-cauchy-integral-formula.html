Комплексный анализ

Интегральная теорема Коши

Фундаментальный результат комплексного анализа: интеграл аналитической функции по замкнутому контуру равен нулю.

Формулировка

Пусть D - односвязная область в комплексной плоскости, f(z) - аналитическая функция в D. Для любого замкнутого кусочно-гладкого контура C ⊂ D:

\[\oint_C f(z) \, dz = 0\]

Интеграл берётся против часовой стрелки.

Условия применимости

1. Аналитичность функции

Функция должна быть аналитической (голоморфной) в области, содержащей контур.

2. Односвязность области

Область не должна иметь «дыр». Для многосвязных областей теорема обобщается.

3. Гладкость контура

Контур должен быть кусочно-гладким и замкнутым.

Интегральная формула Коши

Если f(z) аналитична в D и z₀ ∈ D, то:

\[f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z - z_0} \, dz\]

Более общая формула для производных:

\[f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_C \frac{f(z)}{(z - z_0)^{n+1}} \, dz\]

Геометрическая интерпретация

Аналитическая функция не имеет «циркуляции» в комплексной плоскости. Интеграл по замкнутому пути всегда возвращается в исходную точку.

Аналогия: Как в консервативном силовом поле работа по замкнутому контуру равна нулю.

Применения

1. Вычисление контурных интегралов

Упрощает вычисление сложных интегралов.

2. Теория вычетов

Основа для вычисления интегралов через особые точки.

3. Преобразование Лапласа

Обращение преобразования Лапласа.

4. Квантовая механика

Контурные интегралы в фейнмановских диаграммах.

Пример вычисления

Вычислить \[\oint_C \frac{1}{z} \, dz\] где C - окружность |z| = 1.

Функция f(z) = 1/z аналитична везде кроме z = 0. Так как точка z = 0 внутри контура, теорема Коши неприменима напрямую.

Параметризуем: z = e^(it), dz = ie^(it)dt, t ∈ [0, 2π]:

\[\oint_C \frac{1}{z} \, dz = \int_0^{2\pi} \frac{1}{e^{it}} \cdot ie^{it} \, dt = \int_0^{2\pi} i \, dt = 2\pi i\]

Обобщения

Теорема Коши для многосвязных областей

Интеграл по внешней границе равен сумме интегралов по внутренним границам.

Формула Коши для производных

Аналитическая функция бесконечно дифференцируема.
