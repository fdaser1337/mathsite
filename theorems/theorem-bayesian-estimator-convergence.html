Байесовская статистика

Теорема о сходимости байесовских оценок

Состоятельность и асимптотическая нормальность апостериорного распределения и байесовских оценок.

Основная постановка

Пусть X₁, X₂, ..., Xₙ - независимые одинаково распределённые случайные величины с плотностью p(x|θ), где θ ∈ Θ ⊂ ℝ^k.

Априорное распределение

\[\pi(\theta)\]

Апостериорное распределение

\[\pi_n(\theta|X_1, ..., X_n) = \frac{\pi(\theta)\prod_{i=1}^{n} p(X_i|\theta)}{\int_{\Theta} \pi(\theta')\prod_{i=1}^{n} p(X_i|\theta')d\theta'}\]

Байесовские оценки

  • Апостериорное среднее: E[θ|X₁,...,Xₙ]
  • MAP-оценка: argmax π_n(θ|X₁,...,Xₙ)
  • Апостериорная медиана: median(π_n)

Теорема Бернстайна-фон Мизеса

Формулировка

При определённых условиях регулярности и при n → ∞:

\[\pi_n(\theta|X_1, ..., X_n) \approx \mathcal{N}\left(\hat{\theta}_n, \frac{1}{n}I^{-1}(\hat{\theta}_n)\right)\]

где \hat{θ}_n - оценка максимального правдоподобия, I(θ) - информационная матрица Фишера.

Следствия

  • Апостериорное распределение асимптотически нормально
  • Байесовские оценки асимптотически эквивалентны ММП-оценкам
  • Байесовские доверительные интервалы асимптотически совпадают с частотными

Условия регулярности

Для сходимости байесовских оценок требуются:

  1. Идентифицируемость модели: θ₁ ≠ θ₂ ⇒ p(x|θ₁) ≠ p(x|θ₂)
  2. Регулярность апостериорного: π(θ) > 0 в окрестности истинного θ₀
  3. Дифференцируемость: log p(x|θ) дважды дифференцируема
  4. Неособенность: I(θ₀) положительно определена
  5. Интегрируемость: E[|log p(X|θ)|] < ∞

Теорема состоятельности

Формулировка

При выполнении условий регулярности:

\[\pi_n(\cdot|X_1, ..., X_n) \xrightarrow{P} \delta_{\theta_0}\]

где δ_{θ₀} - вырожденное распределение в точке θ₀.

Интерпретация

Апостериорное распределение сходится к вырожденному в истинном значении параметра.

Доказательство

Используется теорема о сходимости ММП-оценок и локальная аппроксимация апостериорного.

Байесовская центральная предельная теорема

Формулировка

При n → ∞:

\[\sqrt{n}(\theta - \hat{\theta}_n) | X_1, ..., X_n \xrightarrow{d} \mathcal{N}(0, I^{-1}(\theta_0))\]

Это означает, что апостериорное распределение центрировано и масштабировано сходится к стандартному нормальному.

Следствие для байесовских оценок

\[\sqrt{n}(E[\theta|X] - \theta_0) \xrightarrow{d} \mathcal{N}(0, I^{-1}(\theta_0))\]

Примеры

Пример 1: Нормальное распределение с известной дисперсией

X₁, ..., Xₙ ~ N(μ, σ²), σ² известна.

Априорное: μ ~ N(μ₀, τ²).

Апостериорное: μ|X ~ N(μ_n, σ_n²), где:

\[\mu_n = \frac{\frac{1}{\tau^2}\mu_0 + \frac{n}{\sigma^2}\bar{X}}{\frac{1}{\tau^2} + \frac{n}{\sigma^2}}, \quad \sigma_n^2 = \frac{1}{\frac{1}{\tau^2} + \frac{n}{\sigma^2}}\]

Пример 2: Биномиальное распределение

X ~ Binomial(n, p), априорное: p ~ Beta(α, β).

Апостериорное: p|X ~ Beta(α + X, β + n - X).

Пример 3: Распределение Пуассона

X₁, ..., Xₙ ~ Poisson(λ), априорное: λ ~ Gamma(α, β).

Апостериорное: λ|X ~ Gamma(α + ∑Xᵢ, β + n).

Асимптотическая эквивалентность

Сравнение с ММП-оценками

\[E[\theta|X] = \hat{\theta}_n + O_p(n^{-1})\]

Байесовские vs частотные интервалы

Апостериорные доверительные интервалы асимптотически совпадают с доверительными интервалами, основанными на ММП.

Преимущества байесовского подхода

  • Естественный способ учёта априорной информации
  • Корректная обработка малых выборок
  • Единая теория для оценивания и проверки гипотез

Неправильные априорные распределения

Определение

Априорное распределение π(θ) называется неправильным, если ∫π(θ)dθ = ∞.

Примеры

  • π(μ) ∝ 1 для μ ∈ ℝ (равномерное на ℝ)
  • π(σ) ∝ 1/σ для σ > 0 (Джеффрис)

Свойства

При определённых условиях апостериорное распределение остаётся правильным и теоремы сходимости сохраняются.

Вычислительные аспекты

Методы вычисления апостериорного

  • Аналитические методы: сопряжённые априорные распределения
  • Численные методы: квадратуры, аппроксимация
  • Методы Монте-Карло: MCMC, выборка по значимости

MCMC методы

  • Алгоритм Метрополиса-Хастингса
  • Сэмплирование по Гиббсу
  • Гамильтонов Монте-Карло

Модель выбора

Проблема

Как выбрать между различными моделями M₁, M₂, ...?

Байесовский фактор

\[BF_{12} = \frac{P(X|M_1)}{P(X|M_2)} = \frac{\int \pi_1(\theta_1)p(X|\theta_1, M_1)d\theta_1}{\int \pi_2(\theta_2)p(X|\theta_2, M_2)d\theta_2}\]

Асимптотические свойства

При n → ∞ байесовский фактор выбирает истинную модель с вероятностью 1.

Практические применения

1. Машинное обучение

Байесовские нейронные сети, Gaussian Processes.

2. Эконометрика

Векторные авторегрессии, модели панельных данных.

3. Биостатистика

Клинические испытания, эпидемиология.

4. Инженерия

Контроль качества, надёжность.

Историческая справка

Теорема Бернстейна-фон Мизеса предложена в 1940-х годах.

Современная теория развита в работах Лекама, Шварца, Берксона.

Вычислительные методы (MCMC) сделали байесовский подход практически применимым.
