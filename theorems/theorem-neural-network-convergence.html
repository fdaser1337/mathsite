Теорема о сходимости нейронных сетей

Универсальная аппроксимация и сходимость обучения нейронных сетей.

Теорема универсальной аппроксимации

Формулировка (Cybenko, 1989)

Пусть σ: ℝ → ℝ - непрерывная нелинейная функция активации. Тогда для любой непрерывной функции f: [0,1]^n → ℝ и любого ε > 0 существует нейронная сеть с одним скрытым слоем:

\[F(x) = \sum_{i=1}^{N} \alpha_i \sigma(w_i^T x + b_i)\]

такая что |F(x) - f(x)| < ε для всех x ∈ [0,1]^n.

Условия на функцию активации

  • σ непрерывна
  • σ не является полиномом
  • σ ограничена на ℝ

Теорема Хорника (Hornik, 1991)

Обобщение

Нейронные сети с одним скрытым слоем являются плотными в пространстве непрерывных функций на компактных множествах при очень слабых условиях на функцию активации.

Ключевые свойства

  • Не требуется дифференцируемость σ
  • Достаточно, чтобы σ была не полиномиальной и измеримой
  • Результат не зависит от конкретной архитектуры

Сходимость обучения

Постановка задачи

Минимизация эмпирического риска:

\[\min_{\theta} \frac{1}{n}\sum_{i=1}^{n} L(y_i, f_\theta(x_i)) + \lambda R(\theta)\]

где L - функция потерь, R - регуляризатор, λ - параметр регуляризации.

Градиентный спуск

\[\theta_{t+1} = \theta_t - \eta_t \nabla_\theta \left[\frac{1}{n}\sum_{i=1}^{n} L(y_i, f_\theta(x_i)) + \lambda R(\theta)\right]\]

Теорема о сходимости SGD

Условия

Пусть выполнены:

  1. L(·,·) выпукла по второму аргументу
  2. ∇L существует и L-липшицева
  3. Шаги η_t удовлетворяют ∑η_t = ∞, ∑η_t² < ∞
  4. Градиенты ограничены: ||∇L|| ≤ G

Результат

\[\mathbb{E}[L(\bar{\theta}_n)] - L(\theta^*) \leq O\left(\frac{1}{\sqrt{n}}\right)\]

где \bar{θ}_n - среднее параметров, θ* - оптимум.

Скорость аппроксимации

Зависимость от размера сети

Для функций класса Соболева W^{s,p}:

\[\|f - F_N\|_{L^p} \leq C N^{-s/d}\]

где N - число нейронов, d - размерность входа.

Проклятие размерности

Скорость аппроксимации экспоненциально падает с ростом размерности.

Глубокие сети vs мелкие

Теорема о компактном представлении

Некоторые функции, требующие экспоненциального числа нейронов в мелких сетях, могут быть представлены полиномиальным числом нейронов в глубоких сетях.

Примеры

  • Функции с иерархической структурой
  • Композиции простых функций
  • Функции с локальной структурой

Обобщающая способность

Теорема о сложности

Радемахеровская сложность нейронных сетей с W весами:

[\mathfrak{R}_n(\mathcal{F}) \leq O\left(\frac{W\log(n)}{\sqrt{n}}\right)]

Следствие для обобщения

\[L(\hat{f}) - L(f^*) \leq O\left(\frac{W\log(n)}{\sqrt{n}}\right)\]

Примеры функций активации

Sigmoid

\[\sigma(x) = \frac{1}{1 + e^{-x}}\]

ReLU

\[\sigma(x) = \max(0, x)\]

Tanh

\[\sigma(x) = \tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}\]

Все эти функции удовлетворяют условиям теоремы универсальной аппроксимации.

Практические аспекты

Проблемы оптимизации

  • Локальные минимумы
  • Седловые точки
  • Плато
  • Взрывные/исчезающие градиенты

Методы решения

  • Adam, RMSprop, Adagrad
  • Batch normalization
  • Dropout
  • 残差ные соединения

Современные результаты

Теорема о переобучении

В определённых условиях нейронные сети могут идеально запомнить обучающие данные, но при этом хорошо обобщать.

Двойной спуск

Ошибка на тестовых данных сначала возрастает (переобучение), затем убывает (обобщение).

Неявная регуляризация

SGD обладает неявной регуляризирующей способностью.

Применения

1. Компьютерное зрение

Классификация изображений, детекция объектов.

2. Обработка естественного языка

Машинный перевод, анализ тональности.

3. Речевые технологии

Распознавание речи, синтез речи.

4. Рекомендательные системы

Персонализация, предсказание предпочтений.
