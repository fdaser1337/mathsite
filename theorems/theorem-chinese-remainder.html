Теория чисел

Китайская теорема об остатках

Склеивает систему сравнений по взаимно простым модулям в одно сравнение.

Формулировка

Для попарно взаимно простых модулей система сравнений имеет единственное решение по модулю произведения модулей.

\[x\equiv a_i\pmod{m_i},\ i=1,\dots,k\ \Longrightarrow\ x\equiv \sum_{i=1}^k a_i M_i N_i\pmod M,\ \ M_i=\frac{M}{m_i}.\]

Идея доказательства

  1. Для каждого модуля строится специальный базисный элемент.
  2. Каждый базисный элемент дает единицу в своем модуле и ноль в остальных.
  3. Сумма таких элементов дает решение системы; единственность следует из сравнения по произведению модулей.

Следствия и примеры

  • Изоморфизм кольца по произведению попарно взаимно простых модулей.
  • Быстрое восстановление числа из набора остатков.
  • Удобная декомпозиция вычислений по малым модулям.

Применения

  • Криптография и ускорение RSA.
  • Алгоритмы теории чисел.
  • Параллельные вычисления в остаточных классах.
