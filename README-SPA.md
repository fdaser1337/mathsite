# MathLab - SPA Версия

## 🚀 Что изменилось

MathLab превращён из статического сайта с множеством HTML файлов в **Single Page Application (SPA)** с динамической загрузкой контента.

### ✨ Преимущества новой архитектуры:

- **Один HTML файл** вместо десятков отдельных страниц
- **Модульная архитектура JavaScript** с чётким разделением ответственности
- **Динамическая загрузка контента** без перезагрузки страницы
- **Ленивая загрузка** и кэширование для быстрой работы
- **Оффлайн режим** с Service Worker
- **Плавные переходы** между страницами
- **GitHub Pages совместимость** через хеш-роутинг

## 📁 Структура проекта

```
mathsite/
├── index-spa.html          # Главная SPA страница (для GitHub Pages)
├── app.html                 # Альтернативная SPA страница
├── js/
│   ├── app.js               # Главный файл приложения
│   ├── config.js            # Конфигурация
│   ├── state.js             # Управление состоянием
│   └── modules/
│       ├── router.js        # Роутинг и навигация
│       ├── theme.js         # Управление темой
│       ├── search.js        # Поиск
│       ├── theorems.js      # Теоремы
│       ├── bookmarks.js     # Закладки
│       ├── navigation.js    # Навигация
│       ├── exercises.js     # Упражнения
│       ├── visualization.js # Визуализация
│       └── lazy-loader.js   # Ленивая загрузка
├── css/
│   ├── styles.css           # Основные стили
│   └── modules.css          # Стили модулей
├── data/
│   ├── theorems.json        # Данные теорем
│   └── exercises.json       # Данные упражнений
└── sw.js                    # Service Worker для оффлайн режима
```

## 🏗️ Архитектура

### Модули

- **Router** - управляет навигацией и загрузкой страниц
- **ThemeManager** - переключение светлой/тёмной темы
- **SearchManager** - поиск по контенту с подсветкой
- **TheoremManager** - управление теоремами и фильтрами
- **BookmarkManager** - закладки и заметки пользователя
- **ExerciseManager** - интерактивные упражнения
- **VisualizationManager** - построение графиков
- **LazyLoader** - ленивая загрузка и кэширование

### Состояние приложения

Централизованное хранилище `AppState` управляет:
- Темой оформления
- Закладками и заметками
- Прогрессом упражнений
- Поисковой историей
- Фильтрами

## 🎮 Функции

### Горячие клавиши
- `Ctrl + K` - фокус на поиске
- `Ctrl + B` - открыть закладки
- `Ctrl + N` - открыть заметки
- `Ctrl + E` - открыть упражнения
- `Ctrl + V` - открыть визуализацию
- `Ctrl + /` - справка по горячим клавишам

### Интерактивные элементы
- **Поиск** с подсветкой результатов и историей
- **Закладки** с локальным сохранением
- **Упражнения** с мгновенной проверкой
- **Графики** функций с Plotly.js
- **Прогресс-бар** чтения статей
- **Кнопка "Наверх"**

## 🌐 Развертывание

### GitHub Pages
1. Переименуйте `index-spa.html` в `index.html`
2. Загрузите на GitHub Pages
3. Сайт будет работать с хеш-роутингом: `https://username.github.io/mathsite/#!/theorems.html`

### Локальный сервер
```bash
# Для локальной разработки используйте любой HTTP сервер
python -m http.server 8000
# или
npx serve .
```

## 🔧 Конфигурация

Основные настройки в `js/config.js`:

```javascript
export const CONFIG = {
  SEARCH_LIMIT: 10,           // Лимит результатов поиска
  ANIMATION_DURATION: 300,    // Длительность анимаций
  STORAGE_KEYS: {             // Ключи localStorage
    THEME: 'mathlab-theme',
    BOOKMARKS: 'mathlab-bookmarks',
    // ...
  }
};
```

## 📊 Производительность

- **Ленивая загрузка** - контент загружается по мере необходимости
- **Кэширование** - данные и ресурсы кэшируются
- **Service Worker** - оффлайн работа
- **Минимизация HTTP запросов** - динамическая загрузка
- **Оптимизация изображений** - ленивая загрузка

## 🔄 Обновление контента

Контент хранится в JSON файлах в папке `data/`:

- `theorems.json` - все теоремы с метаданными
- `exercises.json` - упражнения с проверкой

Для добавления новой теоремы:
```json
{
  "id": "new-theorem",
  "title": "Название теоремы",
  "description": "Описание",
  "course": "1",
  "difficulty": "basic",
  "sections": {
    "statement": { "content": "Формулировка", "formula": "формула" },
    "proof": { "content": "Доказательство" },
    "examples": []
  }
}
```

## 🐛 Отладка

В консоли браузера доступны:
```javascript
// Статус приложения
window.mathLabApp.getStatus()

// Доступ к модулям
window.mathLabApp.getModule('search')
window.mathLabApp.getModule('router')

// Статистика кэша
window.mathLabApp.getModule('lazyLoader').getCacheStats()
```
