Теорема Больцано-Коши о промежуточном значении

Если непрерывная функция принимает разные значения на концах отрезка, она принимает все промежуточные значения.

Формулировка

Пусть функция f непрерывна на отрезке [a,b] и f(a) ≠ f(b). Тогда для любого числа C между f(a) и f(b) существует точка c ∈ (a,b) такая, что f(c) = C.

\[\forall C \in [f(a), f(b)] \quad \exists c \in (a,b): f(c) = C\]

Геометрическая интерпретация

График непрерывной функции не может «перепрыгнуть» через горизонтальную линию. Он должен пересечь её.

Если f(a) < 0 < f(b), то функция обязательно пересечёт ось Ox.

Доказательство

Рассмотрим случай f(a) < 0 < f(b). Разделим отрезок [a,b] пополам.

Шаг 1: Если f((a+b)/2) = 0, теорема доказана.

Шаг 2: Если f((a+b)/2) > 0, то на [a, (a+b)/2] функция меняет знак.

Шаг 3: Если f((a+b)/2) < 0, то на [(a+b)/2, b] функция меняет знак.

Шаг 4: Продолжаем деление отрезка, получаем стягивающуюся систему отрезков. По лемме о вложенных отрезках существует точка c, где f(c) = 0. ■

Следствия

1. Существование корня

Если непрерывная функция на концах отрезка имеет разные знаки, то внутри отрезка есть хотя бы один корень.

2. Теорема о нуле

Непрерывная функция, не меняющая знака, либо всегда положительна, либо всегда отрицательна.

Применения

  • Численные методы - метод бисекции для решения уравнений
  • Экономика - теорема о существовании равновесия
  • Физика - доказательство существования решений
  • Инженерия - анализ систем

Пример

Доказать, что уравнение x³ - x - 1 = 0 имеет корень на отрезке [1,2].

f(x) = x³ - x - 1 непрерывна. f(1) = -1 < 0, f(2) = 5 > 0.

По теореме Больцано-Коши существует c ∈ (1,2) где f(c) = 0.
