Критерий обратимости матрицы

Формулировка

Для квадратной матрицы эквивалентны: det A ≠ 0, A обратима, rank A = n, столбцы линейно независимы.

Условия

A — квадратная матрица.

Идея доказательства

Эквивалентности выводятся через элементарные преобразования и свойства линейных отображений.

Пример

det [[1,2],[3,4]] = -2, значит матрица обратима.
