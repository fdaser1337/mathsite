Теорема о сходимости бустинга

Сходимость ансамблевых методов и теорема об обучаемости Шапире.

Основная идея бустинга

Бустинг строит сильный классификатор как комбинацию слабых классификаторов.

Ансамбль

\[F(x) = \sum_{t=1}^{T} \alpha_t h_t(x)\]

где h_t(x) - слабые классификаторы, α_t - их веса.

Финальная классификация

\[H(x) = \text{sign}(F(x))\]

AdaBoost алгоритм

Описание

Адаптивный бустинг, который последовательно добавляет классификаторы, фокусируясь на ошибочных примерах.

Веса примеров

\[w_i^{(t+1)} = w_i^{(t)} \exp(-\alpha_t y_i h_t(x_i))\]

Вес классификатора

\[\alpha_t = \frac{1}{2}\ln\left(\frac{1 - \varepsilon_t}{\varepsilon_t}\right)\]

где ε_t - взвешенная ошибка t-го классификатора.

Теорема об обучаемости (Schapire, 1990)

Формулировка

Если слабые классификаторы немного лучше случайных (ε_t < 1/2 - γ), то ансамбль достигает произвольно малой ошибки.

Граница ошибки ансамбля

\[\varepsilon \leq \exp\left(-2\sum_{t=1}^{T} \gamma_t^2\right)\]

где γ_t = 1/2 - ε_t - преимущество слабого классификатора.

Следствие

Если все γ_t ≥ γ > 0, то:

\[\varepsilon \leq \exp(-2T\gamma^2)\]

Теорема о сходимости AdaBoost

Экспоненциальная сходимость

AdaBoost минимизирует экспоненциальную потерю:

\[L(F) = \frac{1}{n}\sum_{i=1}^{n} \exp(-y_i F(x_i))\]

Граница ошибки обучения

\[L_{train}(H) \leq \exp\left(-2\sum_{t=1}^{T} \gamma_t^2\right)\]

Обобщающая способность

При определённых условиях ошибка на тестовых данных также экспоненциально убывает.

Градиентный бустинг

Основная идея

Минимизация произвольной дифференцируемой функции потерь с помощью градиентного спуска в функциональном пространстве.

Алгоритм

\[F_{t+1}(x) = F_t(x) - \eta \nabla_F L(y, F(x))|_{F=F_t}\]

Псевдо-остатки

\[r_{it} = -\left[\frac{\partial L(y_i, F(x_i))}{\partial F(x_i)}\right]_{F=F_t}\]

Теорема о сходимости градиентного бустинга

Условия сходимости

Пусть выполнены:

  1. Функция потерь L выпукла и дифференцируема
  2. Слабые классификаторы ограничены: ||h_t|| ≤ C
  3. Шаг обучения η достаточно мал

Результат

\[L(F_T) - L(F^*) \leq O\left(\frac{1}{T}\right)\]

где F* - оптимальный ансамбль.

Связь с оптимизацией

Аддитивная модель

\[\min_{F} \sum_{i=1}^{n} L(y_i, F(x_i)) + \Omega(F)\]

Жадная оптимизация

Бустинг выполняет жадную пошаговую оптимизацию.

Степенной алгоритм

Обобщение градиентного бустинга с адаптивным выбором шага.

Переобучение бустинга

Ранняя остановка

Остановка обучения при достижении минимума на валидационной выборке.

Регуляризация

  • Ограничение глубины деревьев
  • Уменьшение шага обучения
  • Субдискретизация признаков

Теорема о двойном спуске

При определённых условиях бустинг сначала переобучается, затем улучшает обобщение.

Стамбы решений

  • Случайные подпространства

Теоретические свойства

Маржинальность

\[\rho = \min_i y_i F(x_i)\]

Большая маржинальность соответствует хорошей обобщающей способности.

Граница обобщения

\[R(H) \leq \tilde{O}\left(\sqrt{\frac{d}{n\rho^2}}\right)\]

где d - VC-размерность слабых классификаторов.

Варианты бустинга

LogitBoost

Бустинг с логистической функцией потерь.

Gentle AdaBoost

Модификация AdaBoost с лучшей устойчивостью к шуму.

LPBoost

Линейное программирование формулировка бустинга.

TotalBoost

Регуляризация через ограничение маржинальности.

Практические аспекты

Выбор параметров

  • Количество итераций T
  • Глубина деревьев
  • Шаг обучения η
  • Доля подвыборки

Вычислительная сложность

  • Обучение: O(T × cost(слабый классификатор))
  • Предсказание: O(T × cost(слабый классификатор))

Параллелизация

Возможна параллелизация обучения слабых классификаторов.

Применения

1. Классификация

Кредитный скоринг, детекция мошенничества.

2. Ранжирование

Поисковые системы, рекомендательные системы.

3. Регрессия

Предсказание временных рядов, оценка недвижимости.

4. Обработка естественного языка

Классификация текстов, анализ тональности.

Историческая справка

Теорема об обучаемости доказана Робертом Шапире в 1990 году.

AdaBoost разработан Фройндом и Шапире в 1997 году.

Градиентный бустинг предложен Фридманом в 2001 году.
